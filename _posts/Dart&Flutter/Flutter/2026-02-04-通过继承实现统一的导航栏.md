# 通过继承实现统一的导航栏

统一的导航栏样式，你可以通过创建一个继承自 `StatelessWidget` 或 `StatefulWidget` 的基类来实现类似于 iOS 开发中的 `BaseViewController`。在这个基类中，你可以统一配置页面的背景颜色、导航栏等公共逻辑。示例代码如下：

<font style="color:rgb(64, 64, 64);"></font>

```javascript
import 'package:flutter/material.dart';

/// 基类
class BasePage extends StatelessWidget {
  final String title;
  final Widget child;

  const BasePage({
    Key? key,
    required this.title,
    required this.child,
  }) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text(title),
        backgroundColor: Colors.blue, // 可以在这里统一配置导航栏样式
      ),
      backgroundColor: Colors.grey[200], // 统一背景色
      body: child, // 页面具体内容
    );
  }
}
```

<font style="color:rgb(64, 64, 64);">然后你可以这样使用这个基类，创建子类</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(242, 242, 242);">MyPage</font></code><font style="color:rgb(64, 64, 64);">，返回父类：</font>

```javascript
class MyPage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return BasePage(
      title: lang('我的页面'),
      child: Center(
        child: Text(lang('这是页面内容')),
      ),
    );
  }
}
```

<font style="color:rgb(64, 64, 64);">子类中需要用的组件，在父类里面通过 </font><code><font style="color:rgb(199, 37, 78);background-color:rgb(242, 242, 242);">export</font></code><font style="color:rgb(64, 64, 64);"> 引用：</font>

```javascript
export 'lang_manager.dart';
```

<font style="color:rgb(64, 64, 64);">通过这种方式，你可以把公共配置放在 </font><code><font style="color:rgb(199, 37, 78);background-color:rgb(242, 242, 242);">BasePage</font></code><font style="color:rgb(64, 64, 64);"> 中，而各个页面只需要关注具体的内容。类似于 iOS 的 </font><code><font style="color:rgb(199, 37, 78);background-color:rgb(242, 242, 242);">BaseViewController</font></code><font style="color:rgb(64, 64, 64);">，这种方式实现了代码复用和样式统一。</font>
