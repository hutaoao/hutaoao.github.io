# Html

### DOCTYPE 声明
<font style="color:rgb(68, 68, 68);">推荐使用 HTML5 的文档声明</font>

```html
<!DOCTYPE html>
```

### 页面语言 lang
<font style="color:rgb(68, 68, 68);">推荐使用属性值 </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">cmn-Hans-CN</font><font style="color:rgb(68, 68, 68);">（简体, 中国大陆），但是考虑浏览器和操作系统的兼容性，目前仍然使用 </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">zh-CN</font><font style="color:rgb(68, 68, 68);"> 属性值</font>

```html
<html lang="zh-CN">
```

### 网页编码 charset
<font style="color:rgb(68, 68, 68);">推荐使用 “UTF-8” 编码</font>

```html
<meta charset="UTF-8">
```

### 标签及闭合
+ <font style="color:rgb(51, 51, 51);">尽量标签语义化</font>
+ <font style="color:rgb(51, 51, 51);">标签必须合法且闭合、嵌套正确，标签名需小写</font>
+ <font style="color:rgb(33, 37, 41);">不要在自闭合（self-closing）元素、空元素的尾部添加斜线 — </font>[HTML5 规范](https://dev.w3.org/html5/spec-author-view/syntax.html#syntax-start-tag)<font style="color:rgb(33, 37, 41);"> 中明确说明斜线是可忽略的。</font>
+ <font style="color:rgb(68, 68, 68);">所有具有开始标签和结束标签的元素都要写上起止标签，</font>不要省略可选的结束标签（closing tag）（例如，</li> 或 </body>）。

_<font style="color:#73D13D;">推荐：</font>_

```html
<div>
    <h1>我是h1标题</h1>
    <p>我是一段文字，我有始有终，浏览器能正确解析</p>
</div>
	
<br>
```

_<font style="color:#FA541C;">不推荐：</font>_

```html
<div>
    <h1>我是h1标题</h1>
    <p>我是一段文字，我有始无终，浏览器亦能正确解析
</div>

<br/>
```

### 书写风格
#### 代码大小写
<font style="color:rgb(68, 68, 68);">HTML标签名、类名、标签属性和大部分属性值统一用小写</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<div class="demo"></div>
```

_<font style="color:#FA541C;">不推荐：</font>_

```html
<div class="DEMO"></div>
	
<DIV CLASS="DEMO"></DIV>
```

<font style="color:rgb(68, 68, 68);">HTML文本、CDATA、JavaScript、meta标签某些属性等内容可大小写混合</font>

```html
<!-- 优先使用 IE 最新版本和 Chrome Frame -->
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"/>

<!-- HTML文本内容 -->
<h1>I AM WHAT I AM </h1>

<!-- JavaScript 内容 -->
<script type="text/javascript">
	var demoName = 'demoName';
	...
</script>
	
<!-- CDATA 内容 -->
<script type="text/javascript"><![CDATA[
...
]]></script>
```

> **<font style="color:rgb(77, 77, 77);"><![CDATA[]]> </font>**<font style="color:rgb(77, 77, 77);">此标记用于xml文档中，这个标记所包含的内容将表示为</font>**<font style="color:rgb(77, 77, 77);">纯文本</font>**<font style="color:rgb(77, 77, 77);">，</font><font style="color:rgb(51, 51, 51);">会被解析器忽略。</font>
>

#### 类型属性
<font style="color:rgb(68, 68, 68);">不需要为 CSS、JS 指定类型属性，HTML5 中默认已包含</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<link rel="stylesheet" href="" >
<script src=""></script>
```

_<font style="color:#FA541C;">不推荐：</font>_

```html
<link rel="stylesheet" type="text/css" href="" >
<script type="text/javascript" src="" ></script>
```

#### 元素属性
+ 属性名使用小写字母
+ 属性值使用双引号包围
+ <font style="color:rgb(68, 68, 68);">元素属性值可以写上的都写上</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<input type="text">
	
<input type="radio" name="name" checked="checked" >
```

_<font style="color:#FA541C;">不推荐：</font>_

```html
<input type=text>	
<input type='text'>
	
<input type="radio" name="name" checked >
```

#### [特殊字符引用](#特殊字符引用)
<font style="color:rgb(68, 68, 68);">文本可以和字符引用混合出现。这种方法可以用来转义在文本中不能合法出现的字符。</font>

<font style="color:rgb(68, 68, 68);">在 HTML 中不能使用小于号 “<” 和大于号 “>”特殊字符，浏览器会将它们作为标签解析，若要正确显示，在 HTML 源代码中使用字符实体</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<a href="#">more&gt;&gt;</a>
```

_<font style="color:#FA541C;">不推荐：</font>_

```html

```

#### 代码缩进
<font style="color:rgb(68, 68, 68);">推荐使用四个空格进行代码缩进</font>

```html
<div class="jdc">
    
</div>
```

#### 代码嵌套
<font style="color:rgb(68, 68, 68);">元素嵌套规范，每个块状元素独立一行，内联元素可选</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<div>
    <h1></h1>
    <p></p>
</div>	
<p><span></span><span></span></p>
```

_<font style="color:#FA541C;">不推荐：</font>_

```html
<div>
    <h1></h1><p></p>
</div>	
<p> 
    <span></span>
    <span></span>
</p>
```

<font style="color:rgb(68, 68, 68);">段落元素与标题元素只能嵌套内联元素</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<h1><span></span></h1>
<p><span></span><span></span></p>
```

_<font style="color:#FA541C;">不推荐：</font>_

```html
<h1><div></div></h1>
<p><div></div><div></div></p>
```

### 注释规范
#### 单行注释
<font style="color:rgb(68, 68, 68);">注释内容前后各一个空格字符，注释位于要注释代码的上面，单独占一行</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<!-- 注释内容 -->
<div>...</div>
```

#### 模块注释
<font style="color:rgb(68, 68, 68);">注释内容前后各一个空格字符，</font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);"><!-- S Comment Text --></font><font style="color:rgb(68, 68, 68);"> 表示模块开始，</font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);"><!-- E Comment Text --></font><font style="color:rgb(68, 68, 68);"> 表示模块结束，模块与模块之间相隔一行</font>

_<font style="color:#52C41A;">推荐：</font>_

```html
<!-- S Comment Text A -->	
<div class="mod_a">
    ...
</div>
<!-- E Comment Text A -->
	
<!-- S Comment Text B -->	
<div class="mod_b">
    ...
</div>
<!-- E Comment Text B -->
```

#### 嵌套模块注释
<font style="color:rgb(68, 68, 68);">当模块注释内再出现模块注释的时候，为了突出主要模块，改用</font>

```html
<!-- /Comment Text -->
```

<font style="color:rgb(68, 68, 68);">注释写在模块结尾标签底部，单独一行。</font>

```html
<!-- S Comment Text A -->
<div class="mod_a">
		
    <div class="mod_b">
        ...
    </div>
    <!-- /mod_b -->
    	
    <div class="mod_c">
    	...
    </div>
    <!-- /mod_c -->
		
</div>
<!-- E Comment Text A -->
```
