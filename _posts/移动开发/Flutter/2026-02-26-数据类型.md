# 数据类型

## 常用数据类型

* Numbers (int, double)
* Strings (String)
* Booleans (bool)
* Records ((value1, value2))
* Lists (List, also known as *arrays*)
* Sets (Set)
* Maps (Map)
* Runes (Runes; often replaced by the characters API)
* Symbols (Symbol)



## 字符串类型

### 定义字符串的几种方式

#### 1、var

```dart
var str1 = 'this is str'; // 单引号
var str2 = "this is str"; // 双引号
```

#### 2、String 关键字

```dart
String str1 = 'this is str'; // 单引号
String str2 = "this is str"; // 双引号
```

#### 3、三个引号 - 创建多行字符串

```dart
String str1 = '''this is str
12345
67890
''';

String str1 = """this is str
12345
67890
""";
```

### 字符串的拼接

#### 1、使用$符号

```dart
String str1 = '你好';
String str2 = 'Dart';
print('$str1 $str2'); // 你好 Dart
```

#### 2、使用 + 号

```dart
String str1 = '你好';
String str2 = 'Dart';
print(str1 + ' ' + str2); // 你好 Dart
```

## 数值类型

### num - 同时具有整型和双精度值

```dart
num a = 123;
num b = 123.45;
```

### int - 整型

必须是整型

```dart
int num = 123;
num = 12.34; // 报错
```

### double - 浮点型

既可以是整型，也可以是浮点型

```dart
double num = 12.34;
num = 123;
print(num); // 123.0;
```

## 布尔类型

### 声明

`bool` ：true / false

```dart
bool flag = true;
flag = false;
print(flag); // false
```

### 条件判断

```dart
bool flag = true;
if (flag) {
  print('真'); // 真
} else {
  print('假');
}

var a = 123;
var b = '123';
if (a == b) {
  print('a=b');
} else {
  print('a!=b'); // a!=b
}
```

## List 集合类型

数组/集合 类型

### 定义、获取长度及获取

```dart
var list = ['张三', 20, true];

print(list); // ['张三', 20, true]
print(list.length); // 3
print(list[0]); // 张三
print(list[1]); // 20

// 可以修改长度
list.length = 0;
print(list); // []
```

### 指定类型

```dart
var list1 = <String>['张三', '李四'];
var list2 = <Int>[10, 20];
```

### 增加数据

通过 \[] 创建的集合它的容量是可以变化的

```dart
var list = [];
list.add('张三');
list.add(20);
print(list); // ['张三', 20]
print(list.lenght); 2
```

### 创建一个固定长度的集合 - filled

```dart
var list = List.filled(2, 'hello');
print(list); // ['hello' , 'hello']
print(list.legnth); // 2

list[0] = '张三';
list[1] = '李四';
print(list); // ['张三' , '李四']

list.add('王五'); // 报错；长度固定
list.length = 0; // 报错；长度固定
```

> `new List()`创建数组 只能在老版本使用，新版本不支持

## Maps 类型

### 定义

```dart
var person = {
  "name": '张三',
  "age": 20,
  "work": ['程序员', '外卖员']
};

print(person['name']); // 张三
```

### 通过new创建

```dart
var person = new Map();
p['name'] = '张三';
p['age'] = 20;
p['work'] = ['程序员', '外卖员'];
```

## 类型判断

通过 `is` 关键字来判断类型

```dart
var str = '张三';

if (str is String) {
  print('String');
} else if (str is int) {
  print('int');
} else {
  print('other');
}
```
