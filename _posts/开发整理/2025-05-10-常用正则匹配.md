# 常用正则匹配

### 表单校验只能输入包含0的正整数
```vue
checkInt2(value) {
   const reg = /\d/;
   if (value && reg.test(value)) return this.formData.other = parseInt(value);
   return this.formData.other = value.replace(/[^\d]/g, "");
},
```
