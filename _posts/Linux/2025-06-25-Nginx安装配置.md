# Nginx 安装配置

## <font style="color:rgb(51, 51, 51);">Nginx 安装</font>
<font style="color:rgb(51, 51, 51);">系统平台：</font>CentOS 7.6 x86_64 (64bit)

### <font style="color:rgb(51, 51, 51);">一、安装编译工具及库文件</font>
```bash
[root@localhost src]# yum -y install make zlib zlib-devel gcc-c++ libtool  openssl openssl-devel
```

> yum 下载有问题的请切换源



### <font style="color:rgb(51, 51, 51);">二、首先要安装 PCRE</font>
<font style="color:rgb(51, 51, 51);">PCRE 作用是让 Nginx 支持 Rewrite 功能。</font>

<font style="color:rgb(51, 51, 51);">1、下载 PCRE 安装包，下载地址： </font>[https://downloads.sourceforge.net/project/pcre/pcre/8.45/pcre-8.45.tar.gz](https://downloads.sourceforge.net/project/pcre/pcre/8.45/pcre-8.45.tar.gz)

```bash
[root@localhost src]# cd /usr/local/src/

[root@localhost src]# wget https://downloads.sourceforge.net/project/pcre/pcre/8.45/pcre-8.45.tar.gz
```



<font style="color:rgb(51, 51, 51);">2、解压安装包:</font>

```bash
[root@localhost src]# tar zxvf pcre-8.45.tar.gz
```

<font style="color:rgb(51, 51, 51);">3、进入安装包目录</font>

```bash
[root@localhost src]# cd pcre-8.45
```

<font style="color:rgb(51, 51, 51);">4、编译安装 </font>

```bash
[root@localhost pcre-8.45]# ./configure

[root@localhost pcre-8.45]# make && make install
```

<font style="color:rgb(51, 51, 51);">5、查看pcre版本</font>

```bash
[root@localhost pcre-8.45]# pcre-config --version
```

### <font style="color:rgb(51, 51, 51);">安装 Nginx</font>
<font style="color:rgb(51, 51, 51);">1、下载 Nginx，下载地址：</font>[https://nginx.org/en/download.html](https://nginx.org/en/download.html)

```bash
[root@localhost src]# wget http://nginx.org/download/nginx-1.20.2.tar.gz
```

<font style="color:rgb(51, 51, 51);"> 2、解压安装包</font>

```bash
[root@localhost src]# tar zxvf nginx-1.20.2.tar.gz
```

<font style="color:rgb(51, 51, 51);">3、进入安装包目录</font>

```bash
[root@localhost src]# cd nginx-1.20.2
```

<font style="color:rgb(51, 51, 51);">4、编译安装</font>

```bash
[root@localhost nginx-1.20.2]# ./configure --prefix=/usr/local/webserver/nginx --with-http_stub_status_module --with-http_ssl_module --with-pcre=/usr/local/src/pcre-8.45
```

```bash
[root@localhost nginx-1.20.2]# make && make install
```

<font style="color:rgb(51, 51, 51);">5、查看nginx版本</font>

```bash
[root@localhost nginx-1.20.2]# /usr/local/webserver/nginx/sbin/nginx -v
```

![1646902362693-3521a2ba-0623-41b1-9fdf-85b16f2e9170.png](/assets/img/posts/Linux/1646902362693-3521a2ba-0623-41b1-9fdf-85b16f2e9170-415731.png)

<font style="color:rgb(51, 51, 51);">到此，nginx安装完成。</font>

<font style="color:rgb(51, 51, 51);"></font>

<font style="color:rgb(51, 51, 51);">设置软链接</font>

```bash

ln -s /usr/local/webserver/nginx/conf/ /etc/nginx
```

## <font style="color:rgb(51, 51, 51);">Nginx 其他命令</font>
<font style="color:rgb(51, 51, 51);">以下包含了 Nginx 常用的几个命令：</font>

```bash
/usr/local/webserver/nginx/sbin/nginx -s reload    # 重新载入配置文件
/usr/local/webserver/nginx/sbin/nginx -s reopen    # 重启 Nginx 
/usr/local/webserver/nginx/sbin/nginx -s stop      # 停止 Nginx
```



## 配置单
```shell
events {
    worker_connections  1024;
}


http {
    include       mime.types;
    default_type  application/octet-stream;

    server {
        listen       80;
        server_name  hutaoao.cn;
        #将所有HTTP请求通过rewrite指令重定向到HTTPS。
        rewrite ^(.*)$ https://$host$1;
   }

    # HTTPS server

    server {
        listen       443 ssl;

        server_name  hutaoao.cn;

        ssl_certificate      cert/9715796_hutaoao.cn.pem;
        ssl_certificate_key  cert/9715796_hutaoao.cn.key;

        ssl_session_cache    shared:SSL:1m;
        ssl_session_timeout  5m;

        ssl_ciphers  HIGH:!aNULL:!MD5;
        ssl_prefer_server_ciphers  on;

        location / {
            root   /opt/web-build;
            index  index.html index.htm;
            try_files $uri $uri/ /index.html; #404页面转发
        }

        #代理后台接口
        #匹配到 /api 开头的接口时，转发到下面的服务器地址
        location /api {
            proxy_pass http://111.67.196.29:7001; #转发请求的地址
        }
    }
}
```
