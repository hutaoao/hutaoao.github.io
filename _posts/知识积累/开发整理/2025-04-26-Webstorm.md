# Webstorm

+ 收费
+ 开箱即用、默认集成前端常用插件
+  附带了对 JavaScript、TypeScript、React、React Native、Electron、Vue、Angular、Node.js、HTML、样式表和许多其他技术的开箱即用支持
+ 自动保存
+ 集成Git，branch 管理工具，版本代码对比
+ npm 快捷窗口
+ 自带ESLint 语法检查，帮助规范代码
+ 自动补全代码、检测错误（包括单词错误）和冗余并提供修复建议、声明未使用变量提醒
+ 安全地重构代码：重命名文件、文件夹，会自动修改所有被引用的地方

+ 远程协作开发

WebStorm 包含 [Code With Me](https://www.jetbrains.com/zh-cn/code-with-me/)，这是一款用于远程协作开发和结对编程的服务。 您可以使用该服务**与他人共同进行实时编码**，并直接从 IDE 与您的队友进行交流。



+ ctrl+F（在文件内查找）
+ ctrl+R （在文件内替换）
+ ctrl+X （删除行）
+ ctrl+D （复制行）
+ ctrl+shift+F （指定目录内批量查找）
+ ctrl+shift+R （指定目录内批量替换）
+ ctrl+shift+L （格式化代码）
+ shift+enter（重新开始一行-无论光标在哪个位置）
+ ctrl+shift+N （通过文件名快速查找工程内的文件）
+ ctrl+shift+alt+N （通过一个字符快速查找位置）
