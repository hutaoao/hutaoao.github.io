# 数值扩展

### 二进制和八进制数值的新的写法

<font style="color:rgb(13, 20, 30);">ES6 提供了二进制和八进制数值的新的写法，分别用前缀</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0b</font></code><font style="color:rgb(13, 20, 30);">（或</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0B</font></code><font style="color:rgb(13, 20, 30);">）和</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0o</font></code><font style="color:rgb(13, 20, 30);">（或</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0O</font></code><font style="color:rgb(13, 20, 30);">）表示。</font>

```javascript
0b111110111 === 503 // true
0o767 === 503 // true
```

> <font style="color:rgb(13, 20, 30);">如果要将</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0b</font></code><font style="color:rgb(13, 20, 30);">和</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0o</font></code><font style="color:rgb(13, 20, 30);">前缀的字符串数值转为十进制，要使用</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Number</font></code><font style="color:rgb(13, 20, 30);">方法。</font>

### <font style="color:rgb(237, 28, 36);">数值分隔符</font>

<font style="color:rgb(13, 20, 30);">欧美语言中，较长的数值允许每三位添加一个分隔符（通常是一个逗号），增加数值的可读性。比如，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">1000</font><font style="color:rgb(13, 20, 30);">可以写作</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">1,000</font><font style="color:rgb(13, 20, 30);">。</font>

> <font style="color:rgb(13, 20, 30);">数值分隔符只是一种</font>**<font style="color:#DF2A3F;">书写便利</font>**<font style="color:rgb(13, 20, 30);">，对于 JavaScript 内部数值的存储和输出，并没有影响。</font>

<font style="color:rgb(13, 20, 30);"></font>

#### 允许使用下划线（\_）作为分隔符

<font style="color:rgb(13, 20, 30);">这个数值分隔符没有指定间隔的位数，也就是说，可以每三位添加一个分隔符，也可以每一位、每两位、每四位添加一个。</font>

```javascript
123_00 === 12_300 // true

12345_00 === 123_4500 // true
12345_00 === 1_234_500 // true
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">小数和科学计数法也可以使用数值分隔符。</font>

```javascript
// 小数
0.000_001

// 科学计数法
1e10_000
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">数值分隔符有几个使用</font>**<font style="color:#DF2A3F;">注意点</font>**<font style="color:rgb(13, 20, 30);">。</font>

* <font style="color:rgb(13, 20, 30);">不能放在数值的最前面（leading）或最后面（trailing）。</font>
* <font style="color:rgb(13, 20, 30);">不能两个或两个以上的分隔符连在一起。</font>
* <font style="color:rgb(13, 20, 30);">小数点的前后不能有分隔符。</font>
* <font style="color:rgb(13, 20, 30);">科学计数法里面，表示指数的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">e</font><font style="color:rgb(13, 20, 30);">或</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">E</font><font style="color:rgb(13, 20, 30);">前后不能有分隔符。</font>

<font style="color:rgb(13, 20, 30);">下面的写法都会报错。</font>

```javascript
// 全部报错
3_.141
3._141
1_e12
1e_12
123__456
_1464301
1464301_
```

<font style="color:rgb(13, 20, 30);">下面三个将字符串转成数值的函数，不支持数值分隔符。主要原因是语言的设计者认为，数值分隔符主要是为了编码时书写数值的方便，而不是为了处理外部输入的数据。</font>

* <font style="color:rgb(13, 20, 30);">Number()</font>
* <font style="color:rgb(13, 20, 30);">parseInt()</font>
* <font style="color:rgb(13, 20, 30);">parseFloat()</font>

```javascript
Number('123_456') // NaN
parseInt('123_456') // 123
```

### <font style="color:rgb(237, 28, 36);">Number.parseInt(), Number.parseFloat()</font>

<font style="color:rgb(13, 20, 30);">ES6 将全局方法</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">parseInt()</font></code><font style="color:rgb(13, 20, 30);">和</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">parseFloat()</font></code><font style="color:rgb(13, 20, 30);">，移植到</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Number</font></code><font style="color:rgb(13, 20, 30);">对象上面，行为完全保持不变。</font>

```javascript
// ES5的写法
parseInt('12.34') // 12
parseFloat('123.45#') // 123.45

// ES6的写法
Number.parseInt('12.34') // 12
Number.parseFloat('123.45#') // 123.45
```

<font style="color:rgb(13, 20, 30);">这样做的目的，是逐步减少全局性方法，使得语言逐步模块化。</font>

```javascript
Number.parseInt === parseInt // true
Number.parseFloat === parseFloat // true
```

### Number.isInteger()

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Number.isInteger()</font></code><font style="color:rgb(13, 20, 30);"> 用来判断一个数值是否为整数</font>

```javascript
Number.isInteger(25)   // true
Number.isInteger(25.0) // true

Number.isInteger(25.1) // false
Number.isInteger() 		 // false
Number.isInteger(null) // false
Number.isInteger('15') // false
Number.isInteger(true) // false
```

### Number.EPSILON

<font style="color:rgb(13, 20, 30);">它表示 1 与大于 1 的最小浮点数之间的差。</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Number.EPSILON</font></code><font style="color:rgb(13, 20, 30);">实际上是 JavaScript 能够表示的最小精度。误差如果小于这个值，就可以认为已经没有意义了，即不存在误差了。</font>

<font style="color:rgb(13, 20, 30);">引入一个这么小的量的目的，在于为浮点数计算，设置一个误差范围。我们知道浮点数计算是不精确的。</font>

```javascript
0.1 + 0.2
// 0.30000000000000004

0.1 + 0.2 - 0.3
// 5.551115123125783e-17

5.551115123125783e-17.toFixed(20)
// '0.00000000000000005551'
```

<font style="color:rgb(13, 20, 30);">上面代码解释了，为什么比较</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0.1 + 0.2</font><font style="color:rgb(13, 20, 30);">与</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0.3</font><font style="color:rgb(13, 20, 30);">得到的结果是</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">false</font><font style="color:rgb(13, 20, 30);">。</font>

```javascript
0.1 + 0.2 === 0.3 // false
```

### Math对象扩展

#### Math.trunc

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Math.trunc</font></code><font style="color:rgb(13, 20, 30);"> 方法用于去除一个数的小数部分，返回整数部分。</font>

```javascript
Math.trunc(4.1) // 4
Math.trunc(4.9) // 4
Math.trunc(-4.1) // -4
Math.trunc(-4.9) // -4
Math.trunc(-0.1234) // -0
```

<font style="color:rgb(13, 20, 30);">对于非数值，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Math.trunc</font><font style="color:rgb(13, 20, 30);">内部使用</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Number</font><font style="color:rgb(13, 20, 30);">方法将其先转为数值。</font>

```javascript
Math.trunc('123.456') // 123
Math.trunc(true) //1
Math.trunc(false) // 0
Math.trunc(null) // 0
```

<font style="color:rgb(13, 20, 30);">对于空值和无法截取整数的值，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">NaN</font><font style="color:rgb(13, 20, 30);">。</font>

```javascript
Math.trunc(NaN);      // NaN
Math.trunc('foo');    // NaN
Math.trunc();         // NaN
Math.trunc(undefined) // NaN
```

#### <font style="color:rgb(13, 20, 30);">Math.sign()</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Math.sign</font></code><font style="color:rgb(13, 20, 30);">方法用来判断一个数到底是正数、负数、还是零。对于非数值，会先将其转换为数值。</font>

<font style="color:rgb(13, 20, 30);">它会返回五种值。</font>

* <font style="color:rgb(13, 20, 30);">参数为正数，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">+1</font><font style="color:rgb(13, 20, 30);">；</font>
* <font style="color:rgb(13, 20, 30);">参数为负数，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">-1</font><font style="color:rgb(13, 20, 30);">；</font>
* <font style="color:rgb(13, 20, 30);">参数为 0，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0</font><font style="color:rgb(13, 20, 30);">；</font>
* <font style="color:rgb(13, 20, 30);">参数为-0，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">-0</font><font style="color:rgb(13, 20, 30);">;</font>
* <font style="color:rgb(13, 20, 30);">其他值，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">NaN</font><font style="color:rgb(13, 20, 30);">。</font>

```javascript
Math.sign(-5) // -1
Math.sign(5) // +1
Math.sign(0) // +0
Math.sign(-0) // -0
Math.sign(NaN) // NaN
```

<font style="color:rgb(13, 20, 30);">如果参数是非数值，会自动转为数值。对于那些无法转为数值的值，会返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">NaN</font><font style="color:rgb(13, 20, 30);">。</font>

```javascript
Math.sign('')  // 0
Math.sign(true)  // +1
Math.sign(false)  // 0
Math.sign(null)  // 0
Math.sign('9')  // +1
Math.sign('foo')  // NaN
Math.sign()  // NaN
Math.sign(undefined)  // NaN
```
