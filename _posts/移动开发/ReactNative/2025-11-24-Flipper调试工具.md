# Flipper调试工具

### 下载

`brew install --cask flipper`



### IDB

`brew tap facebook/fb`\
`brew install idb-companion`

> 可能需要进行软链：
> ln -s /opt/homebrew/bin/idb\_companion /usr/local/bin/idb\_companion

`pip3 install fb-idb`

> 此处的pip版本要根据自己的来
