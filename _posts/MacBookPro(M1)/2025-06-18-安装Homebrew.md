# 安装Homebrew

### M1

#### 官方：

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

> 注意：该方法需要科学上网哦

#### 国内：

```bash
/bin/zsh -c "$(curl -fsSL https://gitee.com/huwei1024/HomebrewCN/raw/master/Homebrew.sh)"
```

### intel

```bash
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

<font style="color:rgb(77, 77, 77);">根据自己情况回车运行，按照提示运行下去就可以了</font>

<font style="color:rgb(77, 77, 77);"></font>

<font style="color:rgb(77, 77, 77);">可能会遇到：</font>

```bash
command not found : brew
```

<font style="color:rgb(77, 77, 77);">经过检查后发现，是因为M1芯片的包安装位置不在是以前的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">/usr/local/</font>\ <font style="color:rgb(77, 77, 77);">而是</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">/opt/homebrew</font><font style="color:rgb(77, 77, 77);">，所以要将配置文件里的环境变量改过来</font>

<font style="color:rgb(77, 77, 77);"></font>

<font style="color:rgb(77, 77, 77);">修改环境变量：</font>

```bash
vim ~/.zshrc  //我的Mac终端用的是zsh
```

> 新系统可能不存该文件，可以自行创建  `touch .zshrc`

增加配置：

```bash
export PATH="/opt/homebrew/bin:$PATH"
export PATH="/opt/homebrew/sbin:$PATH"
```

<font style="color:rgb(34, 34, 34);"></font>

<font style="color:rgb(34, 34, 34);">重新执行刚修改的初文件，使之立即生效：</font>

```bash
source ~/.zshrc
```

卸载：

```bash
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/HomebrewUninstall.sh)"
```
