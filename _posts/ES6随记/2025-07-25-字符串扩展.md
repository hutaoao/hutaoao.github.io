# 字符串扩展

### <font style="color:rgb(237, 28, 36);">模板字符串</font>

```javascript
// 字符串中嵌入变量
let name = "Bob", time = "today";
`Hello ${name}, how are you ${time}?`
```

### <font style="color:rgb(237, 28, 36);">includes(), startsWith(), endsWith()</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">indexOf</font></code><font style="color:rgb(13, 20, 30);">方法，可以用来确定一个字符串是否包含在另一个字符串中。ES6 又提供了三种新方法：</font>

* <code>**<font style="color:rgb(13, 20, 30);">includes()</font>**</code><font style="color:rgb(13, 20, 30);">：返回布尔值，表示是否找到了参数字符串。</font>
* <code>**<font style="color:rgb(13, 20, 30);">startsWith()</font>**</code><font style="color:rgb(13, 20, 30);">：返回布尔值，表示参数字符串是否在原字符串的头部。</font>
* <code>**<font style="color:rgb(13, 20, 30);">endsWith()</font>**</code><font style="color:rgb(13, 20, 30);">：返回布尔值，表示参数字符串是否在原字符串的尾部。</font>

```javascript
let s = 'Hello world!';

s.startsWith('Hello') // true
s.endsWith('!') // true
s.includes('o') // true
```

<font style="color:rgb(13, 20, 30);">这三个方法都支持第二个参数，表示开始搜索的位置。</font>

```javascript
let s = 'Hello world!';

s.startsWith('world', 6) // true
s.endsWith('Hello', 5) // true
s.includes('Hello', 6) // false
```

> <code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">endsWith</font></code><font style="color:rgb(13, 20, 30);">的行为与其他两个方法有所不同。它针对前</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">n</font><font style="color:rgb(13, 20, 30);">个字符，而其他两个方法针对从第</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">n</font><font style="color:rgb(13, 20, 30);">个位置直到字符串结束。</font>

### <font style="color:rgb(13, 20, 30);">

</font><font style="color:rgb(237, 28, 36);">repeat() </font> <code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">repeat</font></code><font style="color:rgb(13, 20, 30);">方法返回一个新字符串，表示将原字符串重复</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">n</font><font style="color:rgb(13, 20, 30);">次。</font>

```javascript
'x'.repeat(3) // "xxx"
'hello'.repeat(2) // "hellohello"
'na'.repeat(0) // ""
```

<font style="color:rgb(13, 20, 30);">参数如果是小数，会被取整。</font>

```javascript
'na'.repeat(2.9) // "nana"
```

<font style="color:rgb(13, 20, 30);">如果</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">repeat</font></code><font style="color:rgb(13, 20, 30);">的参数是负数或者</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Infinity</font></code><font style="color:rgb(13, 20, 30);">，会报错。</font>

```javascript
'na'.repeat(Infinity)
// RangeError
'na'.repeat(-1)
// RangeError
```

<font style="color:rgb(13, 20, 30);">但是，如果参数是 0 到-1 之间的小数，则等同于 0，这是因为会先进行取整运算。0 到-1 之间的小数，取整以后等于</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">-0</font><font style="color:rgb(13, 20, 30);">，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">repeat</font><font style="color:rgb(13, 20, 30);">视同为 0。</font>

```javascript
'na'.repeat(-0.9) // ""
```

<font style="color:rgb(13, 20, 30);">参数</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">NaN</font><font style="color:rgb(13, 20, 30);">等同于 0。</font>

```javascript
'na'.repeat(NaN) // ""
```

<font style="color:rgb(13, 20, 30);">如果</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">repeat</font><font style="color:rgb(13, 20, 30);">的参数是字符串，则会先转换成数字。</font>

```javascript
'na'.repeat('na') // ""
'na'.repeat('3') // "nanana"
```

### <font style="color:rgb(237, 28, 36);">padStart()，padEnd()</font>

**<font style="color:rgb(13, 20, 30);">字符串补全长度</font>**<font style="color:rgb(13, 20, 30);">的功能。如果某个字符串不够指定长度，会在头部或尾部补全。</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">padStart()</font></code><font style="color:rgb(13, 20, 30);">用于头部补全，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">padEnd()</font></code><font style="color:rgb(13, 20, 30);">用于尾部补全。</font>

> <code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">padStart()</font></code><font style="color:rgb(13, 20, 30);">和</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">padEnd()</font></code><font style="color:rgb(13, 20, 30);">一共接受两个参数，第一个参数是字符串补全生效的最大长度，第二个参数是用来补全的字符串。</font>

```javascript
'x'.padStart(5, 'ab') // 'ababx'
'x'.padStart(4, 'ab') // 'abax'

'x'.padEnd(5, 'ab') // 'xabab'
'x'.padEnd(4, 'ab') // 'xaba'
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">如果原字符串的长度，等于或大于最大长度，则字符串补全不生效，返回原字符串。</font>

```javascript
'xxx'.padStart(2, 'ab') // 'xxx'
'xxx'.padEnd(2, 'ab') // 'xxx'
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">如果用来补全的字符串与原字符串，两者的长度之和超过了最大长度，则会截去超出位数的补全字符串。</font>

```javascript
'abc'.padStart(10, '0123456789')
// '0123456abc'
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">如果省略第二个参数，默认</font>**<font style="color:#DF2A3F;">使用空格</font>**<font style="color:rgb(13, 20, 30);">补全长度。</font>

```javascript
'x'.padStart(4) // '   x'
'x'.padEnd(4) // 'x   '
```

<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);"></font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">padStart()</font></code><font style="color:rgb(13, 20, 30);">的常见用途是为数值补全指定位数。下面代码生成 10 位的数值字符串。</font>

```javascript
'1'.padStart(10, '0') // "0000000001"
'12'.padStart(10, '0') // "0000000012"
'123456'.padStart(10, '0') // "0000123456"
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">另一个用途是提示字符串格式。</font>

```javascript
'12'.padStart(10, 'YYYY-MM-DD') // "YYYY-MM-12"
'09-12'.padStart(10, 'YYYY-MM-DD') // "YYYY-09-12"
```

### <font style="color:rgb(237, 28, 36);">trimStart()，trimEnd()</font>

<font style="color:rgb(13, 20, 30);">它们的行为与</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trim()</font></code><font style="color:rgb(13, 20, 30);">一致，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trimStart()</font></code><font style="color:rgb(13, 20, 30);">消除字符串头部的空格，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trimEnd()</font></code><font style="color:rgb(13, 20, 30);">消除尾部的空格。它们</font>**<font style="color:#DF2A3F;">返回的都是新字符串，不会修改原始字符串。</font>**

```javascript
const s = '  abc  ';

s.trim() // "abc"
s.trimStart() // "abc  "
s.trimEnd() // "  abc"
```

> <font style="color:rgb(13, 20, 30);">除了空格键，这两个方法对字符串头部（或尾部）的 tab 键、换行符等不可见的空白符号也有效。</font>
>
> <font style="color:rgb(13, 20, 30);">浏览器还部署了额外的两个方法，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trimLeft()</font></code><font style="color:rgb(13, 20, 30);">是</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trimStart()</font></code><font style="color:rgb(13, 20, 30);">的别名，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trimRight()</font></code><font style="color:rgb(13, 20, 30);">是</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">trimEnd()</font></code><font style="color:rgb(13, 20, 30);">的别名。</font>

### <font style="color:rgb(237, 28, 36);">matchAll()</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">matchAll()</font></code><font style="color:rgb(13, 20, 30);">方法返回一个正则表达式在当前字符串的所有匹配</font>

### <font style="color:rgb(237, 28, 36);">replaceAll()</font>

> <font style="color:rgb(13, 20, 30);">它的用法与</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">replace()</font><font style="color:rgb(13, 20, 30);">相同，</font>**<font style="color:#DF2A3F;">返回一个新字符串，不会改变原字符串</font>**<font style="color:rgb(13, 20, 30);">。</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">replace()</font></code><font style="color:rgb(13, 20, 30);">只能替换第一个匹配。</font>

```javascript
'aabbcc'.replace('b', '_')
// 'aa_bcc'
```

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">如果要替换所有的匹配，不得不使用正则表达式的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">g</font><font style="color:rgb(13, 20, 30);">修饰符。</font>

```javascript
'aabbcc'.replace(/b/g, '_')
// 'aa__cc'
```

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">replaceAll()</font></code><font style="color:rgb(13, 20, 30);">方法，可以一次性替换所有匹配。</font>

```javascript
'aabbcc'.replaceAll('b', '_')
// 'aa__cc'
```

<font style="color:rgb(13, 20, 30);">如果</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">searchValue</font></code><font style="color:rgb(13, 20, 30);">是一个不带有</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">g</font><font style="color:rgb(13, 20, 30);">修饰符的正则表达式，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">replaceAll()</font></code><font style="color:rgb(13, 20, 30);">会报错。这一点跟</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">replace()</font></code><font style="color:rgb(13, 20, 30);">不同。</font>

```javascript
// 不报错
'aabbcc'.replace(/b/, '_')

// 报错
'aabbcc'.replaceAll(/b/, '_')
```

### <font style="color:rgb(237, 28, 36);">at()</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">at()</font></code><font style="color:rgb(13, 20, 30);">方法接受一个整数作为参数，返回参数指定位置的字符，支持负索引（即倒数的位置）。</font>

```javascript
const str = 'hello';
str.at(1) // "e"
str.at(-1) // "o"
```

<font style="color:rgb(13, 20, 30);">如果参数位置超出了字符串范围，</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">at()</font></code><font style="color:rgb(13, 20, 30);">返回</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">undefined</font></code><font style="color:rgb(13, 20, 30);">。</font>
