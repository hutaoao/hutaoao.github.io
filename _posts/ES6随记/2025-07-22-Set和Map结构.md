# Set 和 Map 结构

### Set

#### 基本用法

<font style="color:rgb(13, 20, 30);">ES6 提供了新的数据结构 Set。它</font>**<font style="color:rgb(13, 20, 30);">类似于数组</font>**<font style="color:rgb(13, 20, 30);">，但是</font>**<font style="color:rgb(13, 20, 30);">成员的值都是唯一</font>**<font style="color:rgb(13, 20, 30);">的，没有重复的值。</font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font></code><font style="color:rgb(13, 20, 30);">本身是一个构造函数，用来生成 Set 数据结构。</font>

```javascript
const s = new Set();

[2, 3, 5, 4, 5, 2, 2].forEach(x => s.add(x));

for (let i of s) {
  console.log(i);
}
// 2 3 5 4
```

<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);"></font>

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font></code><font style="color:rgb(13, 20, 30);">函数可以接受一个数组作为参数，用来初始化。</font>

```javascript
// 例一
const set = new Set([1, 2, 3, 4, 4]);
[...set]
// [1, 2, 3, 4]

// 例二
const items = new Set([1, 2, 3, 4, 5, 5, 5, 5]);
items.size // 5

// 例三 - 接受 类数组 的对象作为参数
const set = new Set(document.querySelectorAll('div'));
set.size // 56

// 类似于
const set = new Set();
document.querySelectorAll('div').forEach(div => set.add(div));
set.size // 56
```

<font style="color:rgb(13, 20, 30);">上面代码也展示了一种</font>**<font style="color:rgb(13, 20, 30);">去除数组重复成员</font>**<font style="color:rgb(13, 20, 30);">的方法。</font>

```javascript
// 去除数组的重复成员
[...new Set(array)]
```

<font style="color:rgb(13, 20, 30);">上面的方法也可以用于，</font>**<font style="color:rgb(13, 20, 30);">去除字符串里面的重复字符</font>**<font style="color:rgb(13, 20, 30);">。</font>

```javascript
[...new Set('ababbc')].join('')
// "abc"
```

> <font style="color:rgb(13, 20, 30);">向 Set 加入值的时候，不会发生类型转换，所以</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">5</font><font style="color:rgb(13, 20, 30);">和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">"5"</font><font style="color:rgb(13, 20, 30);">是两个不同的值。</font>

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">另外，两个对象总是不相等的。</font>

```javascript
let set = new Set();

set.add({});
set.size // 1

set.add({});
set.size // 2
```

<font style="color:rgb(13, 20, 30);">上面代码表示，由于两个空对象不相等，所以它们被视为两个值。</font>

<font style="color:rgb(13, 20, 30);"></font>

#### <font style="color:rgb(13, 20, 30);">属性和方法</font>

##### 属性

1. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.constructor</font><font style="color:rgb(13, 20, 30);">：构造函数，默认就是</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font><font style="color:rgb(13, 20, 30);">函数。</font>
2. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">size</font>**<font style="color:rgb(13, 20, 30);">：返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font><font style="color:rgb(13, 20, 30);">实例的成员总数。</font>

<font style="color:rgb(13, 20, 30);"></font>

##### 方法 - <font style="color:rgb(13, 20, 30);">操作方法（用于操作数据）</font>

1. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">add</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">(value)</font><font style="color:rgb(13, 20, 30);">：</font>**<font style="color:rgb(13, 20, 30);">添加</font>**<font style="color:rgb(13, 20, 30);">某个值，返回 Set 结构本身。</font>
2. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">delete</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">(value)</font><font style="color:rgb(13, 20, 30);">：</font>**<font style="color:rgb(13, 20, 30);">删除</font>**<font style="color:rgb(13, 20, 30);">某个值，返回一个布尔值，表示删除是否成功。</font>
3. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">has</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">(value)</font><font style="color:rgb(13, 20, 30);">：</font>**<font style="color:rgb(13, 20, 30);">返回一个布尔值</font>**<font style="color:rgb(13, 20, 30);">，表示该值是否为</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font><font style="color:rgb(13, 20, 30);">的成员。</font>
4. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">clear</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：</font>**<font style="color:rgb(13, 20, 30);">清除所有成员</font>**<font style="color:rgb(13, 20, 30);">，没有返回值。</font>

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">简单示例：</font>

```javascript
s.add(1).add(2).add(2);
// 注意2被加入了两次

s.size // 2

s.has(1) // true
s.has(2) // true
s.has(3) // false

s.delete(2);
s.has(2) // false
```

<code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Object</font></code><font style="color:rgb(13, 20, 30);">结构和</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font></code><font style="color:rgb(13, 20, 30);">结构的写法不同。</font>

```javascript
// 对象的写法
const properties = {
  'width': 1,
  'height': 1
};

if (properties[someName]) {
  // do something
}

// Set的写法
const properties = new Set();

properties.add('width');
properties.add('height');

if (properties.has(someName)) {
  // do something
}
```

<code>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Array.from</font>**</code>**<font style="color:rgb(13, 20, 30);">方法可以将 Set 结构转为数组。</font>**

```javascript
const items = new Set([1, 2, 3, 4, 5]);
const array = Array.from(items);
```

<font style="color:rgb(13, 20, 30);">这就提供了</font>**<font style="color:rgb(13, 20, 30);">去除数组重复成员</font>**<font style="color:rgb(13, 20, 30);">的另一种方法。</font>

```javascript
function dedupe(array) {
  return Array.from(new Set(array));
}

dedupe([1, 1, 2, 3]) // [1, 2, 3]
```

##### <font style="color:rgb(13, 20, 30);">方法 - 遍历方法（用于遍历成员）</font>

1. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">keys</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：返回键名的遍历器</font>
2. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：返回键值的遍历器</font>
3. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">entries</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：返回键值对的遍历器</font>
4. <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：使用回调函数遍历每个成员</font>

> <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Set</font><font style="color:rgb(13, 20, 30);">的遍历顺序就是插入顺序。这个特性有时非常有用，比如使用 Set 保存一个回调函数列表，调用时就能保证按照添加顺序调用。</font>

###### <font style="color:rgb(13, 20, 30);">（1）</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">keys()</font><font style="color:rgb(13, 20, 30);">，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values()</font><font style="color:rgb(13, 20, 30);">，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">entries()</font>

<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">keys</font><font style="color:rgb(13, 20, 30);">方法、</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values</font><font style="color:rgb(13, 20, 30);">方法、</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">entries</font><font style="color:rgb(13, 20, 30);">方法返回的都是</font>**<font style="color:rgb(13, 20, 30);">遍历器对象</font>**<font style="color:rgb(13, 20, 30);">。</font>

<font style="color:rgb(13, 20, 30);">由于 Set 结构没有键名，只有键值（或者说</font>**<font style="color:rgb(13, 20, 30);">键名和键值是同一个值</font>**<font style="color:rgb(13, 20, 30);">），所以</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">keys</font><font style="color:rgb(13, 20, 30);">方法和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values</font><font style="color:rgb(13, 20, 30);">方法的行为完全一致。</font>

```javascript
let set = new Set(['red', 'green', 'blue']);

for (let item of set.keys()) {
  console.log(item);
}
// red
// green
// blue

for (let item of set.values()) {
  console.log(item);
}
// red
// green
// blue

for (let item of set.entries()) {
  console.log(item);
}
// ["red", "red"]
// ["green", "green"]
// ["blue", "blue"]
```

<font style="color:rgb(13, 20, 30);">上面代码中，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">entries</font><font style="color:rgb(13, 20, 30);">方法返回的遍历器，同时包括键名和键值，所以每次输出一个数组，</font>**<font style="color:rgb(13, 20, 30);">它的两个成员完全相等</font>**<font style="color:rgb(13, 20, 30);">。</font>

<font style="color:rgb(13, 20, 30);"></font>

**<font style="color:rgb(13, 20, 30);">Set 结构的实例默认可遍历</font>**<font style="color:rgb(13, 20, 30);">，它的默认遍历器生成函数就是它的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values</font><font style="color:rgb(13, 20, 30);">方法。</font>

```javascript
Set.prototype[Symbol.iterator] === Set.prototype.values
// true
```

<font style="color:rgb(13, 20, 30);">这意味着，可以省略</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values</font><font style="color:rgb(13, 20, 30);">方法，直接用</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">for...of</font><font style="color:rgb(13, 20, 30);">循环遍历 Set。</font>

```javascript
let set = new Set(['red', 'green', 'blue']);

for (let x of set) {
  console.log(x);
}
// red
// green
// blue
```

###### <font style="color:rgb(13, 20, 30);">（2）</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach()</font>

<font style="color:rgb(13, 20, 30);">Set 结构的实例与数组一样，也拥有</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach</font><font style="color:rgb(13, 20, 30);">方法，用于对每个成员执行某种操作，没有返回值。</font>

```javascript
let set = new Set([1, 4, 9]);
set.forEach((value, key) => console.log(key + ' : ' + value))
// 1 : 1
// 4 : 4
// 9 : 9
```

<font style="color:rgb(13, 20, 30);">上面代码说明，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach</font><font style="color:rgb(13, 20, 30);">方法的参数就是一个处理函数。该函数的参数与数组的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach</font><font style="color:rgb(13, 20, 30);">一致，依次为键值、键名、集合本身（上例省略了该参数）。</font>

<font style="color:rgb(13, 20, 30);">这里需要注意，</font>**<font style="color:rgb(13, 20, 30);">Set 结构的键名就是键值（两者是同一个值）</font>**<font style="color:rgb(13, 20, 30);">，因此第一个参数与第二个参数的值永远都是一样的。</font>

<font style="color:rgb(13, 20, 30);">另外，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach</font><font style="color:rgb(13, 20, 30);">方法还可以有第二个参数，表示绑定处理函数内部的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">this</font><font style="color:rgb(13, 20, 30);">对象。</font>

###### <font style="color:rgb(13, 20, 30);">（3）遍历的应用</font>

<font style="color:rgb(13, 20, 30);">扩展运算符（</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">...</font><font style="color:rgb(13, 20, 30);">）内部使用</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">for...of</font><font style="color:rgb(13, 20, 30);">循环，所以也可以用于 Set 结构。</font>

```javascript
let set = new Set(['red', 'green', 'blue']);
let arr = [...set];
// ['red', 'green', 'blue']
```

<font style="color:rgb(13, 20, 30);">扩展运算符和 Set 结构相结合，就可以去除数组的重复成员。</font>

```javascript
let arr = [3, 5, 2, 2, 5, 5];
let unique = [...new Set(arr)];
// [3, 5, 2]
```

<font style="color:rgb(13, 20, 30);">而且，数组的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">map</font><font style="color:rgb(13, 20, 30);">和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">filter</font><font style="color:rgb(13, 20, 30);">方法也可以间接用于 Set 了。</font>

```javascript
let set = new Set([1, 2, 3]);
set = new Set([...set].map(x => x * 2));
// 返回Set结构：{2, 4, 6}

let set = new Set([1, 2, 3, 4, 5]);
set = new Set([...set].filter(x => (x % 2) == 0));
// 返回Set结构：{2, 4}
```

### WeakSet

<font style="color:rgb(13, 20, 30);">WeakSet 结构与 Set 类似，也是</font>**<font style="color:rgb(13, 20, 30);">不重复的值的集合</font>**<font style="color:rgb(13, 20, 30);">。但是，它与 Set 有两个区别。</font>

<font style="color:rgb(13, 20, 30);">首先，</font>**<font style="color:rgb(13, 20, 30);">WeakSet 的成员只能是对象，而不能是其他类型的值。</font>**

### Map

#### 含义和基本用法

<font style="color:rgb(13, 20, 30);">JavaScript 的对象（</font>**<font style="color:rgb(13, 20, 30);">Object</font>**<font style="color:rgb(13, 20, 30);">），本质上是</font>**<font style="color:rgb(13, 20, 30);">键值对的集合</font>**<font style="color:rgb(13, 20, 30);">（Hash 结构），但是传统上</font>**<font style="color:rgb(13, 20, 30);">只能用字符串当作键</font>**<font style="color:rgb(13, 20, 30);">。这给它的使用带来了很大的限制。</font>

```javascript
const data = {};
const element = document.getElementById('myDiv');

data[element] = 'metadata';
data['[object HTMLDivElement]'] // "metadata"
```

<font style="color:rgb(13, 20, 30);">上面代码原意是将一个 DOM 节点作为对象</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">data</font><font style="color:rgb(13, 20, 30);">的键，但是由于对象只接受字符串作为键名，所以</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">element</font><font style="color:rgb(13, 20, 30);">被自动转为字符串</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">\[object HTMLDivElement]</font><font style="color:rgb(13, 20, 30);">。</font>

<font style="color:rgb(13, 20, 30);">为了解决这个问题，ES6 提供了 </font><code>**<font style="color:rgb(13, 20, 30);">Map</font>**</code><font style="color:rgb(13, 20, 30);"> 数据结构。它</font>**<font style="color:rgb(13, 20, 30);">类似于对象</font>**<font style="color:rgb(13, 20, 30);">，也是</font>**<font style="color:rgb(13, 20, 30);">键值对的集合</font>**<font style="color:rgb(13, 20, 30);">，</font>**<font style="color:rgb(13, 20, 30);">但是“键”的范围不限于字符串，各种类型的值（包括对象）都可以当作键</font>**<font style="color:rgb(13, 20, 30);">。也就是说，</font>**<font style="color:rgb(13, 20, 30);">Object 结构提供了“字符串—值”的对应，Map 结构提供了“值—值”的对应</font>**<font style="color:rgb(13, 20, 30);">，是一种更完善的 Hash 结构实现。</font>

<font style="color:rgb(13, 20, 30);">如果你需要“键值对”的数据结构，Map 比 Object 更合适。</font>

```javascript
const m = new Map();
const o = {p: 'Hello World'};

m.set(o, 'content')
m.get(o) // "content"

m.has(o) // true
m.delete(o) // true
m.has(o) // false
```

<font style="color:rgb(13, 20, 30);">作为</font>**<font style="color:rgb(13, 20, 30);">构造函数</font>**<font style="color:rgb(13, 20, 30);">，</font><code><font style="color:rgb(13, 20, 30);">Map</font></code><font style="color:rgb(13, 20, 30);"> 也可以接受一个</font>**<font style="color:rgb(13, 20, 30);">数组作为参数</font>**<font style="color:rgb(13, 20, 30);">。该数组的成员是一个个表示键值对的数组。</font>

```javascript
const map = new Map([
  ['name', '张三'],
  ['title', 'Author']
]);

map.size // 2
map.has('name') // true
map.get('name') // "张三"
map.has('title') // true
map.get('title') // "Author"
```

<font style="color:rgb(13, 20, 30);">上面代码在新建 Map 实例时，就指定了两个键</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">name</font><font style="color:rgb(13, 20, 30);">和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">title</font><font style="color:rgb(13, 20, 30);">。</font>

<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Map</font><font style="color:rgb(13, 20, 30);">构造函数接受数组作为参数，实际上执行的是下面的算法：</font>

```javascript
const items = [
  ['name', '张三'],
  ['title', 'Author']
];

const map = new Map();

items.forEach(
  ([key, value]) => map.set(key, value)
);
```

> <font style="color:rgb(13, 20, 30);">注意，只有对同一个对象的引用，Map 结构才将其视为同一个键。这一点要非常小心。</font>
> **<font style="color:#DF2A3F;">Map 的键实际上是跟内存地址绑定的，只要内存地址不一样，就视为两个键</font>**<font style="color:rgb(13, 20, 30);">。这就解决了同名属性碰撞（clash）的问题，我们扩展别人的库的时候，如果使用对象作为键名，就不用担心自己的属性与原作者的属性同名。</font>

```javascript
const map = new Map();

map.set(['a'], 555);
map.get(['a']) // undefined
```

<font style="color:rgb(13, 20, 30);">上面代码的</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">set</font><font style="color:rgb(13, 20, 30);">和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">get</font><font style="color:rgb(13, 20, 30);">方法，表面是针对同一个键，但实际上这是两个不同的数组实例，</font>**<font style="color:rgb(13, 20, 30);">内存地址是不一样的</font>**<font style="color:rgb(13, 20, 30);">，因此</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">get</font><font style="color:rgb(13, 20, 30);">方法无法读取该键，返回</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">undefined</font><font style="color:rgb(13, 20, 30);">。</font>

<font style="color:rgb(13, 20, 30);">如果 Map 的键是一个简单类型的值（数字、字符串、布尔值），则只要两个值严格相等，Map 将其视为一个键，比如</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">0</font><font style="color:rgb(13, 20, 30);">和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">-0</font><font style="color:rgb(13, 20, 30);">就是一个键，布尔值</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">true</font><font style="color:rgb(13, 20, 30);">和字符串</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">true</font><font style="color:rgb(13, 20, 30);">则是两个不同的键。另外，</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">undefined</font><font style="color:rgb(13, 20, 30);">和</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">null</font><font style="color:rgb(13, 20, 30);">也是两个不同的键。虽然</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">NaN</font><font style="color:rgb(13, 20, 30);">不严格相等于自身，但 Map 将其视为同一个键。</font>

```javascript
let map = new Map();

map.set(-0, 123);
map.get(+0) // 123

map.set(true, 1);
map.set('true', 2);
map.get(true) // 1

map.set(undefined, 3);
map.set(null, 4);
map.get(undefined) // 3

map.set(NaN, 123);
map.get(NaN) // 123
```

###### 遍历方法

* <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Map.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">keys</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：返回键名的遍历器。</font>
* <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Map.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">values</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：返回键值的遍历器。</font>
* <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Map.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">entries</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：返回所有成员的遍历器。</font>
* <font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">Map.prototype.</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">forEach</font>**<font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">()</font><font style="color:rgb(13, 20, 30);">：遍历 Map 的所有成员。</font>

> <font style="color:rgb(13, 20, 30);">需要特别注意的是，Map 的遍历顺序就是插入顺序。</font>

<font style="color:rgb(13, 20, 30);"></font>

<font style="color:rgb(13, 20, 30);">Map 结构转为数组结构，比较快速的方法是使用扩展运算符（</font><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">...</font><font style="color:rgb(13, 20, 30);">）。</font>

```javascript
const map = new Map([
  [1, 'one'],
  [2, 'two'],
  [3, 'three'],
]);

[...map.keys()]
// [1, 2, 3]

[...map.values()]
// ['one', 'two', 'three']

[...map.entries()]
// [[1,'one'], [2, 'two'], [3, 'three']]

[...map]
// [[1,'one'], [2, 'two'], [3, 'three']]
```
