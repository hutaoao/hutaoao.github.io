# Linux 切换源

1 centOS7的镜像源文件的目录默认都在该路径下：/etc/yum.repos.d/

```bash
cd /etc/yum.repos.d/
```

2 备份原来的yum镜像源【如果出现错误可以恢复原配置】

```bash
mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak
```

<font style="color:rgb(51, 51, 51);">3 下载</font>**<font style="color:rgb(51, 51, 51);">国内常用的镜像源</font>**<font style="color:rgb(51, 51, 51);">并重命名为CentOS-Base.repo,将其作为当前系统的镜像源</font>

<font style="color:rgb(51, 51, 51);"></font>

**<font style="color:rgb(51, 51, 51);">【阿里云】</font>**

**<font style="color:rgb(51, 51, 51);">CentOS 5</font>**

```bash
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-5.repo
```

**<font style="color:rgb(51, 51, 51);">CentOS 6</font>**

```bash
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-6.repo
```

**<font style="color:rgb(51, 51, 51);">CentOS 7</font>**

```bash
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
```



<font style="color:rgb(51, 51, 51);">4 添加epel源</font>

<font style="color:rgb(51, 51, 51);">安装 epel后，可以通过 yum install package-name安装更多软件。</font>

```bash
wget -P /etc/yum.repos.d/ http://mirrors.aliyun.com/repo/epel-7.repo
```



<font style="color:rgb(51, 51, 51);">5 清理缓存</font>

<font style="color:rgb(51, 51, 51);">yum的缓存文件的存放目录是以下路径下：/var/cache/yum 直接清空给目录，或者使用一下命令均可</font>

```bash
yum clean all
```

<font style="color:rgb(0, 0, 255);"></font>

<font style="color:rgb(51, 51, 51);">6 重新生成缓存</font>

```bash
yum makecache
```

<font style="color:rgb(51, 51, 51);">通过以上操作yum的镜像源就更新完成了。</font><font style="color:rgb(51, 51, 51);">  
</font>
