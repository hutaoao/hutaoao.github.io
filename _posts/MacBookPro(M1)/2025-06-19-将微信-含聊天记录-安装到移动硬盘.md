# 将微信(含聊天记录)安装到移动硬盘

### 说明

<font style="color:rgb(77, 77, 77);">此方法可以实现微信记录&文件全部放在外置硬盘，不占用内置硬盘硬盘空间。</font>

<font style="color:rgb(77, 77, 77);"></font>

### <font style="color:rgb(79, 79, 79);">安装到外置硬盘</font>

1. 下载安装包（dmg包）
2. 双击dmg包，<font style="color:rgb(77, 77, 77);">不要将软件拖到右侧的application里面，将软件直接拖到外置盘的文件夹里 如：</font><code><font style="color:rgb(199, 37, 78);background-color:rgb(249, 242, 244);">/Volumes/T7/Applications</font></code>
3. 登录使用一次：是为了让系统初始化下相关目录（<code><font style="color:#000000;">/Users/xxx/Library/Containers/com.tencent.xinWeChat</font></code>）
4. 将数据文件夹复制到外置硬盘：
   1. 将`/Users/xxx/Library/Containers/com.tencent.xinWeChat`中的`data`整体复制到<font style="color:rgba(0, 0, 0, 0.75);">外置硬盘的指定位置，例如：</font><code><font style="color:#000000;">/Volumes/T7/Library/Containers/com.tencent.xinWeChat</font></code><font style="color:rgba(0, 0, 0, 0.75);">。</font>
   2. <font style="color:rgba(0, 0, 0, 0.75);">移动之后需要把</font><code><font style="color:#000000;">/Users/hutao/Library/Containers/com.tencent.xinWeChat</font></code><font style="color:rgba(0, 0, 0, 0.75);">删除（记住是整个</font>**<font style="color:#000000;">com.tencent.xinWeChat删除</font>**<font style="color:rgba(0, 0, 0, 0.75);">）</font>

### <font style="color:rgb(79, 79, 79);">创建软链接</font>

<code><font style="color:#000000;">ln -s /Volumes/T7/Library/Containers/com.tencent.xinWeChat /Volumes/T7/Library/Containers/com.tencent.xinWeChat</font></code>

<font style="color:#000000;"></font>

### <font style="color:rgb(77, 77, 77);">重新签名微信应用程序</font>

<font style="color:rgb(77, 77, 77);">终端中输入以下命令，并按提示输入管理员密码：</font>

<code><font style="color:rgb(77, 77, 77);">sudo codesign --sign - --force --deep </font><font style="color:#000000;">/Volumes/T7/WeChat.app</font></code>

> <u><font style="color:rgb(77, 77, 77);">这里的路径为外置硬盘上的微信安装路径</font></u>

> <font style="color:rgb(77, 77, 77);">提示：Operation not permitted，解决办法：打开“系统偏好设置 — > 隐私与安全性 — > 完全磁盘访问权限 –> 找到“终端” —> 打开即可</font>
