# Git常用命令

+ 查看所有配置信息：**git config --list**
+ 仓库级(全局)修改用户名：**git config --local(global) user.name 'hutaoao'**
+ 仓库级(全局)修改用户邮箱： **git config --local(global) user.email '332413594@qq.com'**
+ 仓库级(全局)删除用户名：**git config --local(global) --unset user.name**
+ 仓库级(全局)删除用户邮箱：**git config --local(global) --unset user.email**
+ **git clone <地址>**--------------克隆版本库：
+ **git add <file>  **---------------将文件状态加入git暂存区(git add .  会把当前目录下所以文件加入暂存区)
+ **git commit -m ‘注释’**----------将暂存区文件提交
+ **git branch **------------------列出本地分支，并且在当前分支的前面加“*”号标记
+ **git branch -r** ----------------列出远程分支
+ **git branch -a** ----------------列出本地分支和远程分支
+ **git checkout <branch>** -------- 切换分支
+ **git branch <branch>**-----------基于当前分支创建新分支
+ **git checkout -b <branch>** ------创建并切换到新分支
+ **git branch -d <branch>** --------删除本地分支
+ **git tag** ---------------------列出所有tag
+ **git tag <tagName>**  -----------给当前版本打标签
+ **git push origin <tagName>** -----推送到远程仓库
+ **git push origin --tags** ---------推送多个本地标签
