# n切换node版本无效



### 解决方法
  
第一种，修改 N_PREFIX 为 /usr/local  
第二种，删除 N_PREFIX，直接用 export PATH=/usr/local/bin:$PATH

  
第三种，最合理的应该是删除你在用户目录中的 node 安装目录，最简单的是直接删除 /opt/homebrew/bin/node （单独删除这一个文件可能会有不一致问题）
