# call-空安全调用操作符

<code><font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">?.call()</font></code><font style="color:rgb(15, 17, 21);"> 是 Dart 中的</font>**<font style="color:rgb(15, 17, 21);">空安全调用操作符</font>**<font style="color:rgb(15, 17, 21);">，具体作用：</font>

## <font style="color:rgb(15, 17, 21);">作用：</font>

* **<font style="color:rgb(15, 17, 21);">安全调用</font>**<font style="color:rgb(15, 17, 21);">：只有当对象不为</font><font style="color:rgb(15, 17, 21);"> </font><code><font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">null</font></code><font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">时才会调用方法</font>
* **<font style="color:rgb(15, 17, 21);">避免空指针异常</font>**<font style="color:rgb(15, 17, 21);">：如果对象为</font><font style="color:rgb(15, 17, 21);"> </font><code><font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">null</font></code><font style="color:rgb(15, 17, 21);">，整个表达式返回</font><font style="color:rgb(15, 17, 21);"> </font><code><font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">null</font></code><font style="color:rgb(15, 17, 21);">，不会抛出异常</font>

## <font style="color:rgb(15, 17, 21);">用法对比：</font>

### <font style="color:rgb(15, 17, 21);">传统写法（不安全）：</font>

<code><font style="color:rgb(15, 17, 21);">widget.handleConfirmOrg(params);  // 如果 handleConfirmOrg 为 null 会崩溃</font></code>

### <font style="color:rgb(15, 17, 21);">空安全写法：</font>

<code><font style="color:rgb(15, 17, 21);">widget.handleConfirmOrg?.call(params);  // 如果为 null 则什么都不做，不会崩溃</font></code>

## <font style="color:rgb(15, 17, 21);">等价于：</font>

<font style="color:rgb(15, 17, 21);">dart</font>

```dart
if (widget.handleConfirmOrg != null) {
  widget.handleConfirmOrg(params);
}
```

## <font style="color:rgb(15, 17, 21);">在你的代码中的应用：</font>

<font style="color:rgb(15, 17, 21);">dart</font>

```dart
// 回调可能为 null，所以使用安全调用
widget.handleConfirmOrg?.call(params);
widget.handleConfirmBp?.call(slopeId.toString());
```

## <font style="color:rgb(15, 17, 21);">为什么需要：</font>

* <font style="color:rgb(15, 17, 21);">父组件可能没有传递这些回调函数</font>
* <font style="color:rgb(15, 17, 21);">避免因为回调为</font><font style="color:rgb(15, 17, 21);"> </font><code><font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">null</font></code><font style="color:rgb(15, 17, 21);"> </font><font style="color:rgb(15, 17, 21);">导致应用崩溃</font>
* <font style="color:rgb(15, 17, 21);">提高代码的健壮性</font>

<font style="color:rgb(15, 17, 21);">这就是 Dart 空安全特性的重要用法，确保在可能为 </font><code><font style="color:rgb(15, 17, 21);background-color:rgb(235, 238, 242);">null</font></code><font style="color:rgb(15, 17, 21);"> 的情况下安全地调用方法。</font>
