# Flutter常用命令

* `flutter create app_01` ：创建一个新的Flutter项目
* `flutter run`：运行应用程序
* `flutter run -d <deviceId>`：运行指定模拟器或者真机
* `flutter run -d all`：运行所有模拟器（启动完了再运行）
* `flutter devices`：查看计算机上的真机设备和IOS模拟器
* `flutter emulators`：获取模拟器设备列表
* `flutter doctor`：检查flutter开发环境是否正确及提供修复建议
* `flutter pub get`：从pub.dev 下载依赖项。将读取pubspec.yaml文件，并下载所指定的依赖
* `flutter pub upgrade`：更新项目中的依赖
* `flutter clean`：清除项目的构建缓存，删除所有构建输出，并重建项目
* `flutter build apk`：安卓打包
* `flutter build ios --release` ：IOS打包
* `flutter build appbundle` ：打aab包
* `flutter build apk --target-platform android-arm,android-arm64`：打指定 arm arm64的包
* `flutter channel`：查看flutter sdk 分支
* `flutter channel stable`：切换sdk分支到stable
* `flutter` 获取所有指令
* `flutter analye`：分析代码
* `flutter -h`：查看命令帮助信息
* `flutter --version`：查看当前Flutter版本
* `flutter pub outdated`：判断那些过时了的package 依赖以及获取更新建议
* `flutter upgrade v2.2.3`：升级到flutter指定版本
* `flutter downgrade v2.0.3`：回退到flutter指定版本
* `flutter emulator --launch 模拟器名称 `：启动本机的模拟器
* r键：热重载
* R键：重新启动
* q键：退出
* p：显示网络
* P：显示帧率
* o：切换Android与iOS的预览模式
