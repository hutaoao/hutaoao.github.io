# Mac配置多版本JDK

### 使用场景

使用ReactNative开发时：

* 高于<font style="color:rgb(28, 30, 33);">0.73 版本的 React Native 需要 JDK 17 版本，</font>
* <font style="color:rgb(28, 30, 33);">低于 0.73 版本的 React Native 需要 JDK 11 版本，</font>
* <font style="color:rgb(28, 30, 33);">而低于 0.67 的需要 JDK 8 版本。</font>

<font style="color:rgb(28, 30, 33);"></font>

**<font style="color:rgb(28, 30, 33);">JDK是Java开发工具包（Java Development Kit）。</font>**

* <font style="color:rgb(28, 30, 33);">JDK 8 版本于2014年3月18日发布。引入了一系列新功能，如Lambda表达式、函数式接口、Stream API和新的日期/时间API等。</font>
* <font style="color:rgb(28, 30, 33);">JDK 11 版本于2018年9月25日发布。提供了一些重要的改进，如模块化系统、增强的性能、新的HTTP客户端API和局部变量类型推断等。</font>
* <font style="color:rgb(28, 30, 33);">JDK 17 版本于2021年9月14日发布，是Oracle官方长期支持（LTS）版本，提供了一些性能优化、安全性改进、API 更新等方面的调整，以提高 Java 平台的稳定性和可靠性。</font>

<font style="color:rgb(28, 30, 33);"></font>

**<font style="color:rgb(28, 30, 33);">JavaSE、JavaEE和JavaME是Java平台的不同配置和规范。</font>**

* <font style="color:rgb(28, 30, 33);">JavaSE（Java Standard Edition）是Java平台的标准配置，提供了基本的Java编程环境和核心API，适用于通用的桌面、服务器和嵌入式应用程序开发。</font>
* <font style="color:rgb(28, 30, 33);">JavaEE（Java Enterprise Edition）是Java平台的企业级配置，提供了用于开发大型企业应用程序的扩展API和服务，包括Web应用程序开发、企业级数据库访问、消息队列和分布式计算等。</font>
* <font style="color:rgb(28, 30, 33);">JavaME（Java Micro Edition）是Java平台的微型配置，专门用于嵌入式设备和移动设备的应用程序开发，如手机、智能卡和小型传感器等。</font>

<font style="color:rgb(28, 30, 33);">Installer和Compressed Archive区别：</font>

<font style="color:rgb(28, 30, 33);"></font>

<font style="color:rgb(28, 30, 33);">        Installer下载的是一个.dmg可执行文档。</font>

<font style="color:rgb(28, 30, 33);">        Compressed Archive压缩文档，下载的是一个.tar.gz压缩包。</font>

<font style="color:rgb(28, 30, 33);">另外，刚接触Mac的小伙伴你需要知道：</font>

<font style="color:rgb(28, 30, 33);">        DMG文件通常用作Mac应用程序的安装包。是苹果电脑上常见的磁盘映像文件格式，全称为Disk Image。</font>

### <font style="color:rgb(0, 0, 0);">下载安装</font>

<font style="color:rgb(0, 0, 0);">下载对应版本并安装即可。这里使用的是</font>[<font style="color:rgb(0, 0, 0);">Zulu JDK</font>](https://docs.azul.com/core/install/macos)<font style="color:rgb(0, 0, 0);">，其他的JDK也可以。只要是 </font><code><font style="color:rgb(192, 52, 29);background-color:rgb(251, 229, 225);">.pkg</font></code><font style="color:rgb(0, 0, 0);"> 文件，安装后都在 </font><code><font style="color:rgb(192, 52, 29);background-color:rgb(251, 229, 225);">/Library/Java/JavaVirtualMachine/</font></code><font style="color:rgb(0, 0, 0);"> 目录下。</font>

> <font style="color:rgb(0, 0, 0);">可以安装多个 </font>`brew install--cask zulu@11`、`brew install--cask zulu@17`等

vim ~/.zshrc

<font style="color:rgb(0, 0, 0);">这里使用JDK8和JDK17演示，使用vim编辑时，按下i键进入编辑模式，将下面的内容粘贴，按下Esc退出编辑模式，输入 </font><code><font style="color:rgb(192, 52, 29);background-color:rgb(251, 229, 225);">:wq</font></code><font style="color:rgb(0, 0, 0);"> 保存。</font>\ <font style="color:rgb(0, 0, 0);">如果你使用的不是Zulu JDK或者使用的是其他版本，修改对应位置的内容即可。</font>

```typescript
# Java
JAVA_8_HOME=/Library/Java/JavaVirtualMachines/zulu-8.jdk/Contents/Home
JAVA_17_HOME=/Library/Java/JavaVirtualMachines/zulu-17.jdk/Contents/Home
 
alias jdk8="export JAVA_HOME=$JAVA_8_HOME"
alias jdk17="export JAVA_HOME=$JAVA_17_HOME"
 
export JAVA_HOME=$JAVA_17_HOME
export PATH="$JAVA_HOME:$PATH"
```

<font style="color:rgb(0, 0, 0);">更新配置文件</font>

<code><font style="color:rgb(0, 0, 0);">source ~/.zshrc</font></code>

<font style="color:rgb(0, 0, 0);"></font>

### <font style="color:rgb(0, 0, 0);">下载安装（方式二，个人推荐）</font>

本人是把所有软件安装在外置硬盘上（电脑硬盘256根本不够用）：

![1747358046207-25a6d4ad-198c-4ae9-a5cc-1a73661f391c.png](/assets/img/posts/MacBookPro(M1)/1747358046207-25a6d4ad-198c-4ae9-a5cc-1a73661f391c-716096.png)

1. 去[官网](https://www.oracle.com/java/technologies/downloads/archive/)下载，比如下载JDK8，<font style="color:rgb(77, 77, 77);">这里我选择</font>[<font style="color:rgb(78, 161, 219);">Java SE 8 (8u211 and later)</font>](https://www.oracle.com/java/technologies/javase/javase8u211-later-archive-downloads.html)<font style="color:rgb(77, 77, 77);">，点击进入下载页面。</font>可以选择如下图的压缩包：

下载后直接放到自己任何目录解压即可。

![1747358207995-2bca20c8-3e20-4855-86cc-fbae165b5850.png](/assets/img/posts/MacBookPro(M1)/1747358207995-2bca20c8-3e20-4855-86cc-fbae165b5850-897967.png)

2. 配置环境变量，本人 .zshrc 配置如下：

```dart
# Jdk
JAVA_11_HOME=/Volumes/T7/jdk/jdk-11.0.26.jdk/Contents/Home
JAVA_17_HOME=/Volumes/T7/jdk/jdk-17.0.14.jdk/Contents/Home

alias jdk11="export JAVA_HOME=$JAVA_11_HOME"
alias jdk17="export JAVA_HOME=$JAVA_17_HOME"

export JAVA_HOME=$JAVA_11_HOME
export PATH="$JAVA_HOME:$PATH"
```

<font style="color:rgb(0, 0, 0);">更新配置文件</font>

<code><font style="color:rgb(0, 0, 0);">source ~/.zshrc</font></code>

<font style="color:rgb(0, 0, 0);"></font>

**<font style="color:rgb(0, 0, 0);">此时输入 </font>**<code>**<font style="color:rgb(192, 52, 29);background-color:rgb(251, 229, 225);">jdk8</font>**</code>**<font style="color:rgb(0, 0, 0);"> 和 </font>**<code>**<font style="color:rgb(192, 52, 29);background-color:rgb(251, 229, 225);">jdk17</font>**</code>**<font style="color:rgb(0, 0, 0);"> 即可切换版本，默认版本为JDK17。</font>**
