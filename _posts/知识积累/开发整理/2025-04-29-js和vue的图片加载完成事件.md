# js和vue的图片加载完成事件

场景：需要图片在加载过程中显示loading状态

> 语法：`onload` 事件在图片加载完成后立即执行

可以使用`img`的`onload`事件，等图片加载完成之后再执行其他操作

### JS

```html
<body>
  <img
    src="xxx"
    onload="getImg()"
    >
  <script>
    function getImg() {
      console.log('图片加载完成')
    }
  </script>
</body>
```

### vue

```vue
<img
  id="profileImg"
  :src="imgSrc"
  class="profile-img"
  @load="loadImage">
    ...

    
    data() {
      return {
        imgSrc: '',
        imageWidth: 0
      };
    },
    methods: {
      loadImage() {
        console.log('图片加载完成')
      }
    }

```
