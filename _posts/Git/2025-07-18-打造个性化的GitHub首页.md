### <font style="color:rgb(18, 18, 18);">1、Metrics（GitHub 信息统计）</font>
![1664421220287-8fe52667-c9b1-4cfa-a72e-379c58b3b80e.webp](/assets/img/posts/Git/1664421220287-8fe52667-c9b1-4cfa-a72e-379c58b3b80e-753105.webp)

<font style="color:rgb(18, 18, 18);">获得类似上图的 GitHub 数据统计，需要用到一个在线工具</font>[「Metrics」](https://link.zhihu.com/?target=https%3A//metrics.lecoq.io/)<font style="color:rgb(18, 18, 18);">，打开网站之后，在左侧输入你的 GitHub ID，稍等一会，就会返回右侧所有和你相关的数据。</font>

<font style="color:rgb(18, 18, 18);">点击右侧的 Markdown code 选项卡，切换到统计视图对应的 Markdown 链接，复制代码添加到</font><font style="color:rgb(18, 18, 18);background-color:rgb(246, 246, 246);">README.md</font><font style="color:rgb(18, 18, 18);">文件中。</font>

<font style="color:rgb(100, 100, 100);">代码格式可以是markdown语法，也可以是HTML语法，但HTML的扩展性更强一点，建议采用HTML语法格式书写。</font>
