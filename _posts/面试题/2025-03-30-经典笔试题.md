# 经典笔试题

### 让下面代码成立（不该动原代码）
```javascript
var [a, b] = {
	a: 3,
  b: 4,
}
console.log(a, b);
```

答案：

```javascript
Object.prototype[Symbol.iterator] = function() {
  const arr = Object.values(this);
  return arr[Symbol.iterator]();
}

var [a, b] = {
	a: 3,
  b: 4,
}
console.log(a, b);
```

解析：

考察的是 <font style="color:rgb(34, 34, 38);">迭代器（iterator）与可迭代对象（iterable）</font>的理解
