**<font style="color:rgb(0, 0, 0);">解析用户的输入，处理后返回相应的结果</font>**

<font style="color:rgb(88, 88, 88);">会有下面几个属性挂在</font><font style="color:rgb(88, 88, 88);"> </font><font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this</font><font style="color:rgb(88, 88, 88);"> </font><font style="color:rgb(88, 88, 88);">上。</font>

+ <font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this.ctx</font><font style="color:rgb(88, 88, 88);">: 当前请求的上下文 </font>[Context](https://eggjs.org/zh-cn/basics/extend.html#context)<font style="color:rgb(88, 88, 88);"> 对象的实例，通过它我们可以拿到框架封装好的处理当前请求的各种便捷属性和方法。</font>
+ <font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this.app</font><font style="color:rgb(88, 88, 88);">: 当前应用</font><font style="color:rgb(88, 88, 88);"> </font>[Application](https://eggjs.org/zh-cn/basics/extend.html#application)<font style="color:rgb(88, 88, 88);"> </font><font style="color:rgb(88, 88, 88);">对象的实例，通过它我们可以拿到框架提供的全局对象和方法。</font>
+ <font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this.service</font><font style="color:rgb(88, 88, 88);">：应用定义的</font><font style="color:rgb(88, 88, 88);"> </font>[Service](https://eggjs.org/zh-cn/basics/service.html)<font style="color:rgb(88, 88, 88);">，通过它我们可以访问到抽象出的业务层，等价于</font><font style="color:rgb(88, 88, 88);"> </font><font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this.ctx.service</font><font style="color:rgb(88, 88, 88);"> </font><font style="color:rgb(88, 88, 88);">。</font>
+ <font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this.config</font><font style="color:rgb(88, 88, 88);">：应用运行时的</font>[配置项](https://eggjs.org/zh-cn/basics/config.html)<font style="color:rgb(88, 88, 88);">。</font>
+ <font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">this.logger</font><font style="color:rgb(88, 88, 88);">：logger 对象，上面有四个方法（</font><font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">debug</font><font style="color:rgb(88, 88, 88);">，</font><font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">info</font><font style="color:rgb(88, 88, 88);">，</font><font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">warn</font><font style="color:rgb(88, 88, 88);">，</font><font style="color:rgb(104, 113, 104);background-color:rgb(222, 245, 223);">error</font><font style="color:rgb(88, 88, 88);">），分别代表打印四个不同级别的日志，使用方法和效果与 </font>[context logger](https://eggjs.org/zh-cn/core/logger.html#context-logger)<font style="color:rgb(88, 88, 88);"> 中介绍的一样，但是通过这个 logger 对象记录的日志，在日志前面会加上打印该日志的文件路径，以便快速定位日志打印位置。</font>
