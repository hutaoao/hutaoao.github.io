# js判断手机端和pc端

## <font style="color:rgb(79, 79, 79);">Navigator对象：</font>
<font style="color:rgb(77, 77, 77);">navigator对象包含有关浏览器的所有信息，且支持所有浏览器。</font>

<font style="color:rgb(77, 77, 77);">navigator对象有一个属性为userAgent，这是一个只读的</font>[字符串](https://so.csdn.net/so/search?q=%E5%AD%97%E7%AC%A6%E4%B8%B2&spm=1001.2101.3001.7020)<font style="color:rgb(77, 77, 77);">，声明了浏览器用于HTTP请求的用户代理头的值。所以我们可以通过判断navigator.userAgent里面是否包含某些值来判断。如下为userAgent的打印值。</font>

## <font style="color:rgb(79, 79, 79);">方法一：</font>
```javascript
function IsPC(){ 
  var userAgentInfo = navigator.userAgent;
  var Agents = new Array("Android", "iPhone", "SymbianOS", "Windows Phone", "iPad", "iPod");  
  var flag = true;  
  for (var v = 0; v < Agents.length; v++) {  
    if (userAgentInfo.indexOf(Agents[v]) > 0) { flag = false; break; }
  }  
  return flag;
}
```
