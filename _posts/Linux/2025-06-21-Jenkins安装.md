# Jenkins安装

官方文档：

<https://pkg.jenkins.io/redhat-stable/>

### 环境准备

#### <font style="color:rgb(35, 38, 59);">查看系统是否安装过java</font>

`yum list installed | grep java`

#### <font style="color:rgb(35, 38, 59);">查看java软件包列表</font>

`yum list java*`

#### 安装Java

<code><font style="color:#000000;">sudo yum install -y fontconfig java-11-openjdk</font></code>

#### 检查Java版本

`java -version`

### 安装Jenkins

<https://pkg.jenkins.io/redhat-stable/>

使用wget下载Jenkins源，并将其保存到 /etc/yum.repos.d/jenkins.repo

`sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo`

> 证书过期，不检查证书：
>
> <font style="color:#000000;">--no-check-certificate</font>

<code>sudo wget <font style="color:#000000;">--no-check-certificate</font> -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo</code>

使用 rpm 导入 Jenkins 软件包的 GPG 密钥，以确保安装的软件包是经过验证的，并且没有被篡改过

`sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io-2023.key`

使用 yum 安装 Jenkins 软件包

`yum install -y jenkins`

### 启动Jenkins

```shell
service jenkins start
# service jenkins restart restart 重启 Jenkins
# service jenkins restart stop 停止 Jenkins
```

### 给Jenkins放行端口

启动`Jenkins`后，此时`Jenkins`会开启它的默认端口`8080`，但由于防火墙限制，我们需要手动放心`8080`端口才能对外访问。

```shell
firewall-cmd --zone=public --add-port=8080/tcp --permanent
firewall-cmd --zone=public --add-port=50000/tcp --permanent

# 重载入添加的端口
systemctl reload firewalld
```

> 更换端口：
>
> cd <font style="color:#000000;">/usr/lib/systemd/system/jenkins.service</font>
>
> <font style="color:#000000;">重载就OK了</font>
>
> <font style="color:#000000;">systemctl daemon-reload</font>

### 常见命令

1. <font style="color:rgb(51, 51, 51);">重启 Jenkins 服务：</font>`sudo service jenkins restart`
2. <font style="color:rgb(51, 51, 51);">启动 Jenkins 服务：</font><code><font style="color:rgb(51, 51, 51);">sudo service jenkins start</font></code>
3. <font style="color:rgb(51, 51, 51);">停止 Jenkins 服务：</font><code><font style="color:rgb(51, 51, 51);">sudo service jenkins stop</font></code>
4. <font style="color:rgb(51, 51, 51);">查看 Jenkins 进程：在终端中使用 </font><code><font style="color:rgb(51, 51, 51);">ps -ef | grep jenkins</font></code><font style="color:rgb(51, 51, 51);"> 来查找正在运行的 Jenkins 进程。</font>
5. <font style="color:rgb(51, 51, 51);">查看 Jenkins 状态：通过 </font>`service jenkins status` <font style="color:rgb(51, 51, 51);">查看服务的运行状态。</font>

### 权限问题

1.将 jenkins 账号加入到 root 组中。

`gpasswd -a jenkins root`

2.修改/etc/sysconfig/jenkins文件，添加如下配置。

`JENKINS_USER="root"`

`JENKINS_GROUP="root"`

3.重启 Jenkins

4.验证

`groups jenkins`
