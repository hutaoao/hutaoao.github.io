# Css

### 命名
+ <font style="color:rgb(51, 51, 51);">id 采用驼峰式命名</font>
+ <font style="color:rgb(51, 51, 51);">类名使用小写字母，以中划线分隔</font>
+ <font style="color:rgb(51, 51, 51);">scss 中的变量、函数、混合、placeholder 采用驼峰式命名</font>



### <font style="color:rgb(68, 68, 68);">代码格式化</font>
<font style="color:rgb(68, 68, 68);">使用展开格式书写样式</font>

```css
.jdc {
    display: block;
    width: 50px;
}
```

### <font style="color:rgb(68, 68, 68);">代码大小写</font>
<font style="color:rgb(68, 68, 68);">样式选择器，属性名，属性值关键字全部使用小写字母书写，属性字符串允许使用大小写。</font>

```css
.jdc {
	display:block;
}
```

### <font style="color:rgb(68, 68, 68);">选择器</font>
+ <font style="color:rgb(68, 68, 68);">尽量少用通用选择器</font><font style="color:rgb(68, 68, 68);"> </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">*</font>
+ <font style="color:rgb(68, 68, 68);">尽量不使用 ID 选择器</font>
+ <font style="color:rgb(68, 68, 68);">不使用无具体语义定义的标签选择器</font>

```css
.jdc {}
.jdc li {}
.jdc li p {}
```

### <font style="color:rgb(68, 68, 68);">代码缩进</font>
<font style="color:rgb(68, 68, 68);">使用四个空格进行代码缩进</font>

```css
.jdc {
    width: 100%;
    height: 100%;
}
```

### <font style="color:rgb(68, 68, 68);">分号</font>
```css
.jdc {
    width: 100%;
    height: 100%;
}
```

### <font style="color:rgb(68, 68, 68);">代码易读性</font>
<font style="color:rgb(68, 68, 68);">左括号与类名之间一个空格，冒号与属性值之间一个空格</font>

```css
.jdc { 
    width: 100%; 
}
```

属性名 与之后的 : 之间不允许包含空格， : 与 属性值 之间必须包含空格

```css
margin: 0;
```

<font style="color:rgb(68, 68, 68);">逗号分隔的取值，逗号之后一个空格</font>

```css
.jdc {
    box-shadow: 1px 1px 1px #333, 2px 2px 2px #ccc;
}
```

<font style="color:rgb(68, 68, 68);">为单个css选择器或新申明开启新行</font>

```css
.jdc, 
.jdc_logo, 
.jdc_hd {
    color: #ff0;
}
.nav{
    color: #fff;
}
```

<font style="color:rgb(68, 68, 68);">颜色值 </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">rgb()</font><font style="color:rgb(68, 68, 68);"> </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">rgba()</font><font style="color:rgb(68, 68, 68);"> </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">hsl()</font><font style="color:rgb(68, 68, 68);"> </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">hsla()</font><font style="color:rgb(68, 68, 68);"> </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">rect()</font><font style="color:rgb(68, 68, 68);"> 中不需有空格，且取值不要带有不必要的 0</font>

```css
.jdc {
    color: rgba(255,255,255,.5);
}
```

<font style="color:rgb(68, 68, 68);">属性值十六进制数值能用简写的尽量用简写</font>

```css
.jdc {
    color: #fff;
}
```

<font style="color:rgb(68, 68, 68);">不要为 </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">0</font><font style="color:rgb(68, 68, 68);"> 指明单位</font>

```css
.jdc {
    margin: 0 10px;
}
```

### <font style="color:rgb(68, 68, 68);">属性值引号</font>
<font style="color:rgb(68, 68, 68);">css属性值需要用到引号时，使用单引号</font>

```css
.jdc { 
	font-family: 'Hiragino Sans GB';
}
```

### <font style="color:rgb(68, 68, 68);">属性书写顺序</font>
<font style="color:rgb(68, 68, 68);">建议遵循以下顺序：</font>

1. <font style="color:rgb(68, 68, 68);">布局定位属性：display / position / float / clear / visibility / overflow</font>
2. <font style="color:rgb(68, 68, 68);">自身属性：width / height / margin / padding / border / background</font>
3. <font style="color:rgb(68, 68, 68);">文本属性：color / font / text-decoration / text-align / vertical-align / white- space / break-word</font>
4. <font style="color:rgb(68, 68, 68);">其他属性（CSS3）：content / cursor / border-radius / box-shadow / text-shadow / background:linear-gradient …</font>

```css
.jdc {
    display: block;
    position: relative;
    float: left;
    width: 100px;
    height: 100px;
    margin: 0 10px;
    padding: 20px 0;
    font-family: Arial, 'Helvetica Neue', Helvetica, sans-serif;
    color: #333;
    background: rgba(0,0,0,.5);
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    -o-border-radius: 10px;
    -ms-border-radius: 10px;
    border-radius: 10px;
}
```

### <font style="color:rgb(68, 68, 68);">CSS3浏览器私有前缀写法</font>
<font style="color:rgb(68, 68, 68);">CSS3 浏览器私有前缀在前，标准前缀在后</font>

```css
.jdc {
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    -o-border-radius: 10px;
    -ms-border-radius: 10px;
    border-radius: 10px;
}
```

### 注释规范
#### <font style="color:rgb(68, 68, 68);">单行注释</font>
<font style="color:rgb(68, 68, 68);">注释内容第一个字符和最后一个字符都是一个空格字符，单独占一行，行与行之间相隔一行</font>

```css
/* Comment Text */
.jdc {
		display: block;
}
```

#### <font style="color:rgb(68, 68, 68);">模块注释</font>
<font style="color:rgb(68, 68, 68);">注释内容第一个字符和最后一个字符都是一个空格字符，</font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">/*</font><font style="color:rgb(68, 68, 68);"> 与 模块信息描述占一行，多个横线分隔符</font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">-</font><font style="color:rgb(68, 68, 68);">与</font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">*/</font><font style="color:rgb(68, 68, 68);">占一行，行与行之间相隔两行</font>

```css
/* Module A
---------------------------------------------------------------- */
.mod_a {}


/* Module B
---------------------------------------------------------------- */
.mod_b {}
```

#### <font style="color:rgb(68, 68, 68);">文件信息注释</font>
<font style="color:rgb(68, 68, 68);">在样式文件编码声明 </font><font style="color:rgb(77, 77, 76);background-color:rgb(238, 238, 238);">@charset</font><font style="color:rgb(68, 68, 68);"> 语句下面注明页面名称、作者、创建日期等信息</font>

```css
@charset "UTF-8";
/**
 * @desc File Info
 * @author Author Name
 * @date 2021-06-18
 */
```
