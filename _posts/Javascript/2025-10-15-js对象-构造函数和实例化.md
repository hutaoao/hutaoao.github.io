# js 对象、构造函数和实例化

### <font style="color:rgb(0, 0, 0);">1. 对象</font>

<code><font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">JavaScript</font></code><font style="color:rgb(0, 0, 0);"> 中对象是一种引用数据类型，除了原始值，其他值皆为对象。</font>

<font style="color:rgb(0, 0, 0);">对象是一种包含属性和方法的数据，以 </font><code><font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">key-value</font></code><font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);"> </font><font style="color:rgb(0, 0, 0);">形式存在。</font>

```javascript
const obj = {name: "tom", age: 23} 
```

### <font style="color:rgb(0, 0, 0);">2. 构造函数</font>

<font style="color:rgb(0, 0, 0);">构造函数不同于一般函数，它可以使用 </font><code><font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">new</font></code> <font style="color:rgb(0, 0, 0);">关键字创建并初始化对象。</font>

**<font style="color:rgb(75, 75, 75);">定义：</font>\*\*\*\*<font style="color:#F5222D;">通过 new 函数名来实例化对象的函数叫构造函数。任何的函数都可以作为构造函数存在</font>**<font style="color:rgb(75, 75, 75);">。之所以有构造函数与普通函数之分，主要从功能上进行区别的，构造函数的主要功能为初始化对象，特点是和new 一起使用。new就是在创建对象，从无到有，构造函数就是在为初始化的对象添加属性和方法。</font>**<font style="color:rgb(75, 75, 75);">构造函数定义时首字母大写</font>**<font style="color:rgb(75, 75, 75);">（规范）。</font>

**<font style="color:rgb(75, 75, 75);">对new理解：</font>**<font style="color:rgb(75, 75, 75);">new 申请内存，创建对象，当调用new时，后台会隐式执行new Object()创建对象。所以，通过new创建的字符串、数字是引用类型，而是非值类型</font>

```javascript
function Person(name, age) {
    this.name = name;
    this.age = age;
}
```

### <font style="color:rgb(0, 0, 0);">3. 实例化</font>

<font style="color:#F5222D;">创建一个对象，生成一个具体实例，便是对象的实例化。</font>

**<font style="color:rgb(0, 0, 0);">通过内置 </font>\*\*\*\*<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">Object</font>** **<font style="color:rgb(0, 0, 0);">创建一个对象实例。</font>**

```javascript
var person = new Object();
person.name = "tom";
person.age = 23;
person.sayHi = function() {
    console.log("Hello World!");
}
```

**<font style="color:rgb(0, 0, 0);">通过字面量 </font>\*\*\*\*<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">{}</font>** **<font style="color:rgb(0, 0, 0);">创建一个对象实例。</font>**

```javascript
var person = {
    name: "tom",
    age: 23,
    sayHi: function() {
        console.log("Hello World!");
    }
}
```

**<font style="color:rgb(0, 0, 0);"></font>**

**<font style="color:rgb(0, 0, 0);">通过构造函数创建一个对象实例。</font>**

```javascript
function Person(name, age) {
    this.name = name;
    this.age = age;
    this.sayHi = function() {
        console.log("Hello World!");
    }
}

var person = new Person("tom", 23);
```
