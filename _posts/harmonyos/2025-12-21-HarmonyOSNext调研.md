# HarmonyOS Next调研

### <font style="color:rgb(34, 34, 34);">前言</font>
<font style="color:rgb(34, 34, 34);">10月22日举行“原生鸿蒙之夜暨华为全场景新品发布会”</font>

<font style="color:rgba(0, 0, 0, 0.9);">重点介绍了</font>**<font style="color:rgba(0, 0, 0, 0.9);">HarmonyOS NEXT</font>**<font style="color:rgba(0, 0, 0, 0.9);">也就是鸿蒙操作系统·原生，华为称其为面向万物智联时代的全场景智能操作系统。</font>

<font style="color:rgba(0, 0, 0, 0.9);"></font>

### <font style="color:rgba(0, 0, 0, 0.9);">特点</font>
+ 使用全新的开发语言（**ArkTS**）、全新的UI框架（**ArkUI**）、专属开发工具（**<font style="color:rgb(0, 0, 0);">DevEco Studio</font>**）；
+ 不再兼容Android（apk）格式，仅支持**HAP**格式（须上架才能安装使用）；
+ 不支持热更新服务；

### <font style="color:rgba(0, 0, 0, 0.9);">传统应用（APP）和元服务</font>
<font style="color:rgba(0, 0, 0, 0.9);">HarmonyOS除支持传统的需要安装的应用（以下简称传统应用）外，还支持更加方便快捷的免安装的应用，即元服务。</font>

<font style="color:rgba(0, 0, 0, 0.9);">元服务是HarmonyOS提供的一种轻量应用程序形态，具备秒开直达，纯净清爽；服务相伴，恰合时宜；即用即走，账号相随；一体两面，嵌入运行；原生智能，全域搜索；高效开发，生而可信等特征。</font>

<font style="color:rgba(0, 0, 0, 0.9);">元服务可独立上架、分发、运行，独立实现业务闭环，可大幅提升信息与服务的获取效率。</font>
