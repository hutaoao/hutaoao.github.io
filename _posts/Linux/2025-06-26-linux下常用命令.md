# linux下常用命令

#### <font style="color:rgb(0, 0, 0);">一、打开对外开放端口号</font>
<font style="color:rgb(0, 0, 0);">1、查看对外开放的端口状态  
</font><font style="color:rgb(0, 0, 0);">查询已开放的端口 netstat -ntulp | grep 端口号：可以具体查看某一个端口号  
</font><font style="color:rgb(0, 0, 0);">查询指定端口是否已开 firewall-cmd --query-port=666/tcp  
</font><font style="color:rgb(0, 0, 0);">提示 yes，表示开启；no表示未开启。</font>

<font style="color:rgb(0, 0, 0);">  
</font><font style="color:rgb(0, 0, 0);">2、查看防火墙状态  
</font><font style="color:rgb(0, 0, 0);">查看防火墙状态 systemctl status firewalld  
</font><font style="color:rgb(0, 0, 0);">开启防火墙 systemctl start firewalld  
</font><font style="color:rgb(0, 0, 0);">关闭防火墙 systemctl stop firewalld  
</font><font style="color:rgb(0, 0, 0);">开启防火墙 service firewalld start  
</font><font style="color:rgb(0, 0, 0);">若遇到无法开启  
</font><font style="color:rgb(0, 0, 0);">先用：systemctl unmask firewalld.service  
</font><font style="color:rgb(0, 0, 0);">然后：systemctl start firewalld.service  
</font>

<font style="color:rgb(0, 0, 0);">  
</font><font style="color:rgb(0, 0, 0);">3、对外开发端口  
</font><font style="color:rgb(0, 0, 0);">查看想开的端口是否已开：firewall-cmd --query-port=6379/tcp  
</font><font style="color:rgb(0, 0, 0);">添加指定需要开放的端口：firewall-cmd --add-port=123/tcp --permanent  
</font><font style="color:rgb(0, 0, 0);">重载入添加的端口：firewall-cmd --reload  
</font><font style="color:rgb(0, 0, 0);">查询指定端口是否开启成功：firewall-cmd --query-port=123/tcp</font>

<font style="color:rgb(0, 0, 0);">移除指定端口：firewall-cmd --permanent --remove-port=123/tcp</font>

#### <font style="color:rgb(0, 0, 0);">三、zip、unzip压缩和解压缩</font>
<font style="color:rgb(0, 0, 0);">1.zip 用法---压缩文件</font>

```bash
zip [参数] [打包后的文件名] [打包的目录路径]
```

##### <font style="color:rgb(0, 0, 0);">常用参数：</font>
<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-a</font><font style="color:rgb(0, 0, 0);"> 将文件转成ASCII模式</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-F</font><font style="color:rgb(0, 0, 0);"> 尝试修复损坏的压缩文件</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-h</font><font style="color:rgb(0, 0, 0);"> 显示帮助界面</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-m</font><font style="color:rgb(0, 0, 0);"> 将文件压缩之后，删除源文件</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-n</font><font style="color:rgb(0, 0, 0);"> 特定字符串 不压缩具有特定字尾字符串的文件</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-o</font><font style="color:rgb(0, 0, 0);"> 将压缩文件内的所有文件的最新变动时间设为压缩时候的时间</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-q</font><font style="color:rgb(0, 0, 0);"> 安静模式，在压缩的时候不显示指令的执行过程</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-r</font><font style="color:rgb(0, 0, 0);"> 将指定的目录下的所有子目录以及文件一起处理</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-S</font><font style="color:rgb(0, 0, 0);"> 包含系统文件和隐含文件（S是大写）</font>

<font style="color:rgb(0, 0, 0);">示例：将指定目录/tmp压缩成test.zip文件</font>

```bash
zip -r test.zip tmb/
```

<font style="color:rgb(0, 0, 0);">2.unzip用法  -解压缩</font>

```bash
unzip [参数] [待解压缩文件]
```

<font style="color:rgb(0, 0, 0);">在linux下解压zip文件，最简单的方式就是unzip命令直接跟上要解压的zip文件。</font>

<font style="color:rgb(0, 0, 0);background-color:rgb(245, 245, 245);">unzip [待解压缩文件]</font>

###### <font style="color:rgb(0, 0, 0);">常用参数：</font>
<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-n</font><font style="color:rgb(0, 0, 0);"> 解压缩时不要覆盖原有的文件；</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-o</font><font style="color:rgb(0, 0, 0);"> 不必先询问用户，unzip执行后覆盖原有的文件；</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-P [密码]</font><font style="color:rgb(0, 0, 0);"> 使用zip的密码选项；</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-q</font><font style="color:rgb(0, 0, 0);"> 执行时不显示任何信息；</font>

<font style="color:rgb(232, 62, 140);background-color:rgb(246, 246, 246);">-d [目录]</font><font style="color:rgb(0, 0, 0);"> 指定文件解压缩后所要存储的目录；</font>

<font style="color:rgb(0, 0, 0);">示例：  zip -r xxx.zip  /temp             unzip -d /temp   /usr/local/test.zip</font>
