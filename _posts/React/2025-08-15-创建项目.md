# 创建项目

### 创建基于TS的react项目（无需全局安装脚手架）

`npx create-react-app my-app --template typescript`

## create-react-app 工程，修改react端口号

* window环境：

set PORT=5000 && react-scripts start



* mac环境：

PORT=5000 react-scripts start



> 竟然区分环境！！！
