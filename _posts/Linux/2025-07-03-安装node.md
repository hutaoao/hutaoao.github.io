# 安装node

yum安装

```bash
[root@localhost ~]# yum -y install nodejs
```

查看版本

```bash
[root@localhost ~]# node -v
```

> npm 找不到，很神奇
>



**<font style="color:#E8323C;">推荐</font>**

<font style="color:rgb(51, 51, 51);">Node 官网已经把 linux 下载版本更改为已编译好的版本了，我们可以直接下载解压后使用：</font>

<font style="color:rgb(51, 51, 51);">下载</font>

```bash
[root@localhost software]# wget https://nodejs.org/dist/v16.14.0/node-v16.14.0-linux-x64.tar.xz
```

解压

```bash
[root@localhost software]# tar xf node-v16.14.0-linux-x64.tar.xz
```

修改下目录名

```bash
[root@localhost software]# mv node-v16.14.0-linux-x64 node
```

<font style="color:rgb(51, 51, 51);">解压文件的 bin 目录底下包含了 node、npm 等命令，我们可以使用 ln 命令来设置软连接：</font>

```bash
[root@localhost ~]# ln -s /usr/boftware/node/bin/node /usr/bin/node

[root@localhost ~]# ln -s /usr/software/node/bin/npm /usr/bin/npm
```

查看版本

```bash
[root@localhost ~]# node -v 

[root@localhost ~]# npm -v 
```
